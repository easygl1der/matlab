#### 4.3 符号绘图 ⭐⭐
```matlab
【易错题】
x = 1:0.1:2; x = sym(x);
fplot(x, x.^2) 为何无法绘图？

答案：fplot需要符号函数或匿名函数，不是采样点数组
正确：fplot(@(x) x.^2, [1, 2])

```

【函数对比】
- ez系列（ezplot, ezmesh, ezsurf）：老版的符号可视化函数，只需给出符号表达式，无需采样，自动采样和选区间。新版本不推荐，可能警告。
  - 例子：
    ```matlab
    syms x y
    ezplot(sin(x))            % 绘制y=sin(x)的2D曲线
    ezmesh(x^2 + y^2)         % 绘制z=x^2+y^2的三维网格面
    ezsurf(x^2 + y^2)         % 绘制z=x^2+y^2的三维彩色曲面
    ```

- 新推荐：fplot, fsurf, fmesh——功能更强、定制灵活。支持符号表达式、匿名函数句柄等，可自动采样与区间设置。fplot适合2D曲线，fsurf画三维曲面，fmesh画三维网格曲面。
  - 例子：
    ```matlab
    fplot(@(x) sin(x), [0, 2*pi])                         % 2D曲线：y=sin(x)
    fsurf(@(x, y) x.^2 + y.^2, [-1 1 -1 1])               % 3D曲面：z=x^2+y^2
    fmesh(@(x, y) sin(x).*cos(y), [-pi pi -pi pi])        % 3D网格：z=sin(x)cos(y)
    ```

- plot, mesh, surf等常规函数：需要先手动准备“数组数据”，自行采样更灵活但易出错，并不适合直接绘制符号表达式。
  - 例子：
    ```matlab
    x = linspace(0, 2*pi, 100);
    y = sin(x);
    plot(x, y);                      % 手动采样二维曲线

    [X, Y] = meshgrid(-1:0.05:1, -1:0.05:1);
    Z = X.^2 + Y.^2;
    mesh(X, Y, Z);                   % 3D网格曲面
    surf(X, Y, Z);                   % 3D彩色曲面
    ```

### 题型1：矩阵操作解读（6分）⭐⭐⭐

#### 样题1：基础矩阵运算
```matlab
clear
A = [1,1,4; 1,3,2; 1,4,5];
B = A .* A          % B = [1 1 16; 1 9 4; 1 16 25]
C = sum(A)          % C = [3, 8, 11]
D = det(A)          % D = 8
E = A(6)            % E = 4 (按列优先，第6个元素)
F = max(min(A))     % F = 2 (每列最小值的最大值)
```

**解题要点**:
1. 区分 `*` (矩阵乘) 和 `.*` (逐元素乘)
2. `sum(A)` 按列求和 → 行向量
3. 单下标按列优先顺序
4. 嵌套函数：从内到外

#### 样题2：矩阵变换
```matlab
clear
A = reshape(1:9, 3, 3)   % A = [1 4 7; 2 5 8; 3 6 9]
B = flipud(A)            % B = [9 6 3; 8 5 2; 7 4 1]
C = tril(A)              % C = [1 0 0; 2 5 0; 3 6 9]
D = mean(A, 2)           % D = [4; 5; 6] 按行求均值
E = cond(tril(C))        % E = 9 条件数
```

**解题要点**:
1. `reshape` 按列填充
2. `flipud` 上下翻转，`fliplr` 左右翻转
3. `tril` 下三角，`triu` 上三角
4. `mean(A, 2)`   mean(A,1) 是按列（每列一个均值），mean(A,2) 是按行（每行一个均值）。

### 题型2：向量化改写（6分）⭐⭐⭐ 必考

#### 核心考点：去除for循环

**样题**：设A为m×n矩阵，向量化改写
```matlab
% 原代码（低效）
for i = 1:m
    for j = 1:n
        if (i < j)
            A(i,j) = 0;
        elseif (i == j)
            A(i,j) = 1;
        else
            A(i,j) = 2*A(i,j);
        end
    end
end

% 向量化答案（多种）
A = tril(A);                              % 去掉上三角
A = 2*A;                                  % 全部乘2
A(1:m+1:min(m*m, m*n)) = 1;              % 对角元为1

% 或
A = tril(A, -1)*2 + eye(min(m,n));       % 下三角×2 + 对角1

% 或使用sub2ind
idx_diag = sub2ind([m,n], 1:min(m,n), 1:min(m,n));
A = tril(A) * 2;
A(idx_diag) = 1;
```

**评分标准** (6分):
- 正确处理两个部分：3-4分
- 正确处理一个部分：1-2分
- 代码有误酌情给分
- 减少一层for也有分

**常见技巧**:
1. `tril`/`triu`: 提取三角阵
2. `diag`: 处理对角元
3. 逻辑索引：`A(condition) = value`
4. `sub2ind`: 双下标→单下标

#### 样题2：ODE数值解
```matlab
clear, d = 1e-5; x = 0:d:2;
y = zeros(2, length(x)); 
y(:,1) = [0; 1];                    % 两个初值

for i = 2:length(x)
    y(:,i) = y(:,i-1) + d*(x(i-1) - y(:,i-1));
end
```

### 题型4：动态规划-背包问题（10分）⭐⭐⭐ 新重点

#### 问题描述
- n件商品，前m件为特殊商品
- 每件价格c_i元，性能值v_i
- 预算C元 (C≤2000)
- **约束**: 至多只能购买一件特殊物品
- **目标**: 最大化总性能值

#### 数学模型
$$\max \sum_{i=1}^n x_i v_i$$
$$\text{s.t.} \quad \sum_{i=1}^n x_i c_i \leq C, \quad \sum_{i=1}^m x_i \leq 1, \quad x_i \in \{0,1\}$$

#### 动态规划算法

**状态定义**: 
- $F(i, j)$ = 前i件商品，花费j元时的最大性能值

**状态转移**:
- 特殊商品($i \leq m$): 至多选1件
  $$F(i, j) = \begin{cases}
  F(i-1, j) & \text{if } j < c_i \\
  \max(v_i, F(i-1, j)) & \text{if } j \geq c_i
  \end{cases}$$

- 普通商品($i > m$): 可选可不选
  $$F(i, j) = \begin{cases}
  F(i-1, j) & \text{if } j < c_i \\
  \max(F(i-1, j), F(i-1, j-c_i) + v_i) & \text{if } j \geq c_i
  \end{cases}$$

#### 完整代码

```matlab
% 动态规划-背包问题（特殊物品至多选1件）
function solve_knapsack()
    % 读取数据（示例）
    % load('Q31.mat');  % m, n, C, c, v
    % 或使用测试数据
    n = 6; m = 3; C = 20;
    c = [5, 10, 15, 18, 1, 1];
    v = [10, 20, 30, 4, 4, 4];
    
    % 1. 初始化DP表
    F = zeros(n, C+1);      % F(i,j): 前i件，花费j元的最大性能
    
    % 2. 处理第一件特殊商品
    F(1, 1:c(1)) = 0;               % 买不起
    F(1, c(1)+1:C+1) = v(1);        % 买得起，获得v(1)
    
    % 3. 处理剩余特殊商品 (i=2 to m)
    for i = 2:m
        F(i, 1:c(i)) = F(i-1, 1:c(i));     % 买不起第i件
        
        % 买得起：只能用v(i)替换之前的（至多选1件）
        F(i, c(i)+1:C+1) = max(v(i), F(i-1, c(i)+1:C+1));
    end
    
    % 此时F(m,:)表示前m件特殊商品至多选1件的最优值
    
    % 4. 处理普通商品 (i=m+1 to n)
    for i = m+1:n
        F(i, 1:c(i)) = F(i-1, 1:c(i));     % 买不起
        
        % 买得起：选或不选
        F(i, c(i)+1:C+1) = max(F(i-1, 1:C-c(i)+1) + v(i), ...
                                F(i-1, c(i)+1:C+1));
    end
    
    % 5. 输出结果
    max_value = F(n, C+1);
    fprintf('最大总性能值为 %d。\n', max_value);
end
```

### 算法1：改进欧拉法 ⭐⭐⭐ 必考

**适用**: 一阶ODE初值问题 $y' = f(x,y), y(x_0)=y_0$

**算法**:
1. 预测步：$\tilde{y}_{i+1} = y_i + h f(x_i, y_i)$
2. 校正步：$y_{i+1} = y_i + \frac{h}{2}[f(x_i, y_i) + f(x_{i+1}, \tilde{y}_{i+1})]$

**代码模板**:
```matlab
clear, d = 1e-5;                % 步长
x = x_start:d:x_end;            % 区间
y = zeros(1, length(x));
yp = @(X,Y) 右端函数;            % f(x,y)
y(1) = 初值;

for i = 2:length(x)
    temp = y(i-1) + d*yp(x(i-1), y(i-1));
    y(i) = y(i-1) + d/2*(yp(x(i-1),y(i-1)) + yp(x(i),temp));
end

Result = y(end);
```

**考试要点**:
- 必须会默写
- 记住两步公式
- 注意初值位置 `y(1) = y0`
- 最后必须赋值 `Result = ...`

### 算法2：手写梯形公式 ⭐⭐⭐ 必考

**适用**: 计算定积分 $\int_a^b f(x)dx$

**公式**:
$$\int_a^b f(x)dx \approx h \sum_{i=1}^{n-1} \frac{f(x_i) + f(x_{i+1})}{2}$$

**代码模板**:
```matlab
clear, d = 步长;
x = a:d:b;                              % 采样点
fx = f(x);                              % 函数值
fm = (fx(1:end-1) + fx(2:end))/2;      % 梯形高度
Result = sum(fm) * d;                   % 积分值
```

**关键技巧**:
- `fx(1:end-1)` 和 `fx(2:end)` 计算相邻均值
- 不能使用 `trapz` 函数！

### 算法3：二分法求根 ⭐⭐⭐

**适用**: f(x)=0，已知f(a)·f(b)<0

**原理**: 区间不断二分，保留有根的一半

**代码模板**:
```matlab
f = @(x) 方程表达式;
a = 左端点; b = 右端点;     % 确保f(a)*f(b)<0
err = 误差限度;             % 如1e-5

while (b-a) > err
    c = (a+b)/2;
    if f(c) > 0
        b = c;              % 或根据f(a)符号判断
    elseif f(c) < 0
        a = c;
    else
        break;              % f(c)=0，找到精确根
    end
end

Result = c;                 % 或(a+b)/2
```

### 算法5：三分法 ⭐⭐

**适用**: 单峰函数求极值

**原理**: 比较两个三分点，舍弃1/3区间

**代码模板**:

```matlab
    alpha1 = a*2/3 + b*1/3;      % 左三分点

    if f(alpha1) > f(alpha2)     % 求最小值
        a = alpha1;               % 舍弃左侧

    end
```

### 算法6：高斯消元法（选主元）⭐⭐

**适用**: 求解Ax=b

**关键**: 列选主元，向量化消元

```matlab
function x = gauss_elim(A, b)
    n = size(A, 1);
    Ab = [A, b];
    
    % 前向消元
    for k = 1:n-1
        % 选主元
        [~, pivot_row] = max(abs(Ab(k:n, k)));
        pivot_row = pivot_row + k - 1;
        
        % 行交换
        if pivot_row ~= k
            Ab([k, pivot_row], :) = Ab([pivot_row, k], :);
        end
        
        % 向量化消元
        rows_below = k+1:n;
        factor = Ab(rows_below, k) / Ab(k, k);
        Ab(rows_below, k:end) = Ab(rows_below, k:end) - ...
                                factor * Ab(k, k:end);
    end
    
    % 回代
    x = zeros(n, 1);
    for i = n:-1:1
        x(i) = (Ab(i,end) - Ab(i,i+1:n)*x(i+1:n)) / Ab(i,i);
    end
end
```

### 算法7：黄金分割法 ⭐⭐

**适用**: 单峰函数一元优化

**黄金分割比**: $\phi = \frac{\sqrt{5}-1}{2} \approx 0.618$

```matlab
f = @(x) 目标函数;
a = 0; b = 1; tol = 1e-6;
phi = (sqrt(5)-1)/2;

x1 = b - phi*(b-a);
x2 = a + phi*(b-a);
f1 = f(x1); f2 = f(x2);

while (b-a) > tol
    if f1 < f2          % 求最小值
        b = x2; x2 = x1; f2 = f1;
        x1 = b - phi*(b-a);
        f1 = f(x1);
    else
        a = x1; x1 = x2; f1 = f2;
        x2 = a + phi*(b-a);
        f2 = f(x2);
    end
end

x_min = (a+b)/2;
```

### 算法8：牛顿迭代法 ⭐⭐

#### 求根: f(x)=0
```matlab
f = @(x) 函数;
df = @(x) 导数;
x0 = 初始值;
tol = 1e-6;

while true
    x_new = x0 - f(x0)/df(x0);
    if abs(x_new - x0) < tol
        break;
    end
    x0 = x_new;
end

root = x0;
```

#### 求极值: min f(x)
```matlab
df = @(x) 一阶导;
ddf = @(x) 二阶导;
x0 = 初始值;

while true
    x_new = x0 - df(x0)/ddf(x0);
    if abs(x_new - x0) < tol
        break;
    end
    x0 = x_new;
end

x_min = x0;
```

## ⚠️ 高频易错点（考前必看）

### 错误1：符号计算忘记声明
```matlab
❌ int(x^2, 0, 1)      % 错误！x未定义
✅ syms x; int(x^2, 0, 1)
```

### 错误2：矩阵运算符混淆
```matlab
❌ A = [1,2]; B = [3;4]; C = A*B     % 维度不匹配
✅ C = A * B'  或  C = A .* B'
```

### 错误3：向量化运算忘记点
```matlab
❌ x = [1,2,3]; y = sin(x)/x        % 错误！
✅ y = sin(x)./x                     % 逐元素除
```

### 错误4：索引方向错误
```matlab
sum(A)       % 按列求和 → 行向量 [1×n]
sum(A, 2)    % 按行求和 → 列向量 [m×1]
```

### 错误5：步长选择不当
```matlab
❌ d = 1e-16;   % 太小！接近机器精度，数值不稳定
❌ d = 0.1;     % 太大！精度不够
✅ d = 1e-5;    % 合适
```

### 错误6：忘记最后赋值
```matlab
❌ y(end)              % 只显示，没赋值
✅ Result = y(end);    % 必须赋值给Result
```

### 错误7：fft vs fourier
```matlab
【填空题陷阱】
问：fft是什么变换？
❌ 傅里叶变换                    % 不完整！
```

### 错误8：循环变量冲突
```matlab

```

## 📖 重点定理与公式

### 数值积分
```
梯形公式: ∫f(x)dx ≈ h·Σ[f(xᵢ)+f(xᵢ₊₁)]/2
中矩形: ∫f(x)dx ≈ h·Σf((xᵢ+xᵢ₊₁)/2)
辛普森: ∫f(x)dx ≈ (h/6)·Σ[f(xᵢ)+4f(x̄ᵢ)+f(xᵢ₊₁)]
```

### 数值微分
```
向前差分: f'(x) ≈ [f(x+h)-f(x)]/h, O(h)精度
中心差分: f'(x) ≈ [f(x+h)-f(x-h)]/(2h), O(h²)精度
```

### ODE数值解
```
欧拉法: yᵢ₊₁ = yᵢ + h·f(xᵢ, yᵢ), O(h)精度
改进欧拉: yᵢ₊₁ = yᵢ + (h/2)[f(xᵢ,yᵢ) + f(xᵢ₊₁,ỹᵢ₊₁)], O(h²)精度
```

### 矩阵范数
```
‖A‖₁ = max_j Σᵢ|aᵢⱼ| (列和范数)
‖A‖₂ = σₘₐₓ(A) (谱范数，最大奇异值)
‖A‖∞ = max_i Σⱼ|aᵢⱼ| (行和范数)
‖A‖F = √(Σᵢⱼ aᵢⱼ²) (Frobenius范数)
```
