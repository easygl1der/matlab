## 一、向量化操作的重要性（复习）[^1]

### 多层循环效率问题[^1]

```matlab
% 低效率方法：使用两层循环逐个处理元素
AR = [1,3; 2,4];
AI = [5,7; 6,8];
A = AR - AI*i;

% 两层循环，效率低，速度慢！！
for m = 1:2
    for n = 1:2
        Am1(m,n) = abs(A(m,n));
        Aa1(m,n) = angle(A(m,n))*180/pi;  % 幅角
    end
end
% 结果正确，但耗时较长

% 高效率方法：向量化操作
Am2 = abs(A)                    % 一步计算所有元素的模
Aa2 = angle(A)*180/pi          % 一步计算所有元素的幅角

% 对于大型矩阵，性能差异可达100倍以上！
```

**关键原则**：MATLAB采用解释性执行系统，不利于多层for循环操作，但使用数组（向量、矩阵）化编程可以简洁而且高效。

## 二、一维数组（向量）[^1]

### 向量的创建方法[^1]

#### 方法1：冒号操作符（步长法）

```matlab
% 基本格式：start:step:end

% 递增向量
A = 1:5
A = 1    2    3    4    5

% 出现小数（超过终点值时停止）
A = 1:4.5               % 仍以1为步长递增，超过4.5结束
A = 1    2    3    4

% 递减向量
A = 5:-1:1              % 起始值5，步长-1，终值1
A = 5    4    3    2    1

% 空向量（起始值大于终值且步长为正）
A = 5:1                 % 没有指定步长（默认为1），返回空向量
A = 空的 1×0 double 行矢量

% 自定义步长
A = 0:pi/4:pi
A = 0    0.7854    1.5708    2.3562    3.1416

% 递减且自定义步长
A = 1:-0.1:0
A = 1.0000  0.9000  0.8000  0.7000  0.6000  0.5000 ...
    0.4000  0.3000  0.2000  0.1000  0
```


#### 方法2：linspace函数（指定点数）

```matlab
% 基本格式：linspace(start, end, num_points)
% 在[start, end]上生成num_points个等距点
% 等价于：start:(end-start)/(num_points-1):end

b1 = linspace(0, pi, 4)
b1 = 0    1.0472    2.0944    3.1416
% 生成4个点，均匀分布在[0, π]上

% 自动计算步长
step = (pi - 0) / (4 - 1)     % step = π/3
```


#### 方法3：logspace函数（对数均匀分布）

```matlab
% 基本格式：logspace(a, b, n)
% 构成等比数列：10^a, 10^(a + (b-a)/(n-1)), ..., 10^b
% 等价于：10.^(a:(b-a)/(n-1):b)

b2 = logspace(0, 3, 4)
b2 = 1    10    100    1000

% 手动验证
logspace_manual = 10.^(0 : (3-0)/(4-1) : 3)
logspace_manual = 1.0000e+00  1.0000e+01  1.0000e+02  1.0000e+03
```

## 五、矩阵的形态改变[^1]

### reshape：改变矩阵尺寸[^1]

```matlab
% 按照相同的排列顺序重新排列元素（从左到右逐列，从上到下）
a = 1:8
a = 1    2    3    4    5    6    7    8

% 4×2矩阵
A = reshape(a, 4, 2)
A =
    1    5
    2    6
    3    7
    4    8

% 改为2×4矩阵（元素顺序按列优先）
A = reshape(A, 2, 4)
A =
    1    3    5    7
    2    4    6    8

% 关键原理：列优先排列
% MATLAB按列存储矩阵，reshape按列优先重排
```

```matlab
C = fliplr(A)

```

### 单序号编址[^1]

```matlab
% MATLAB按列优先的顺序进行单序号编址

%   i = k - (j-1)*m

Ai1r = 1    3    18

Ai1c = A(ind')       % 返回列向量（ind'为列向量）

    1
    3
    18
```

## 八、数组的逐元素运算[^1]

### 基本算术运算[^1]

```matlab
% 数组运算与矩阵运算的区别
A = [1, 2, 3; 4, 5, 6];
B = [1, 1, 1; 2, 2, 2];

% 数组加减法（与矩阵相同）
A + B                 % 对应元素相加
A - B                 % 对应元素相减

% 数组乘法（.*：对应元素相乘）
C = A .* B
C =
    1    2    3
    8   10   12

% 对比矩阵乘法（*）
A * B'                % 这是矩阵乘法，维度必须匹配

% 数组除法（./和.\）
D = A ./ B            % A的每个元素除以B的对应元素
E = A .\ B            % B的每个元素除以A的对应元素（顺序相反！）

% 数组幂运算（.^）
F = A .^ 2            % 每个元素平方
F =
```

### 低效方法：使用for循环和if语句[^1]

### 高效方法：数组化编程[^1]

```matlab
function y = exm030303_2(x)
% 高效的向量化实现方法

% 使用逻辑数组标记三个分段的位置
L1 = x <= -1;              % 第一段：x <= -1
L2 = (-1 < x) & (x <= 1);  % 第二段：-1 < x <= 1  
L3 = 1 < x;                % 第三段：x > 1

% 为结果矩阵申请存储空间
y = zeros(size(x));

% 分段赋值（无for和if）
y(L1) = x(L1);                              % 第一段：y = x
y(L2) = x(L2).^3 .* cos(2*pi*x(L2));       % 第二段：y = x³*cos(2πx)
y(L3) = exp(-x(L3) + 1);                   % 第三段：y = e^(-x+1)
end

% 🔑 关键问题：y(L1) = x(L1) 这种语法为什么有意义？

% 具体数值例子演示逻辑索引的工作原理
x_demo = [-2, -1, 0, 1, 2];  % 测试向量

% 步骤1：创建逻辑数组（这是关键！）
L1 = x_demo <= -1           % [true, true, false, false, false]
L2 = (-1 < x_demo) & (x_demo <= 1)  % [false, false, true, true, false]
L3 = 1 < x_demo             % [false, false, false, false, true]

% 步骤2：理解逻辑索引的含义
% L1 = [true, true, false, false, false] 意思是：
% "选择第1个和第2个元素，忽略第3、4、5个元素"

% 验证逻辑索引的提取效果：
x_demo(L1)                  % 结果：[-2, -1] （只提取L1为true的位置）
x_demo(L2)                  % 结果：[0, 1]   （只提取L2为true的位置）
x_demo(L3)                  % 结果：[2]      （只提取L3为true的位置）

% 步骤3：初始化结果
y_demo = zeros(size(x_demo))  % [0, 0, 0, 0, 0]

% 步骤4：逻辑索引赋值的工作原理
% y_demo(L1) = x_demo(L1) 的含义：
% "把 x_demo 中 L1=true 的元素，赋值给 y_demo 中 L1=true 的位置"

y_demo(L1) = x_demo(L1)      
% 等价于：y_demo([1,2]) = x_demo([1,2]) 
% 即：y_demo(1) = x_demo(1), y_demo(2) = x_demo(2)
% 结果：y_demo = [-2, -1, 0, 0, 0]

y_demo(L2) = x_demo(L2).^3 .* cos(2*pi*x_demo(L2))  
% 等价于：y_demo([3,4]) = [0^3*cos(0), 1^3*cos(2π)]
% 结果：y_demo = [-2, -1, 0, -1, 0]

y_demo(L3) = exp(-x_demo(L3) + 1)  
% 等价于：y_demo(5) = exp(-2+1) = exp(-1) ≈ 0.3679
% 结果：y_demo = [-2, -1, 0, -1, 0.3679]

% 🎯 核心理解：
% L1 是一个"选择器"，告诉MATLAB"我要操作哪些位置"
% y(L1) 表示"y数组中L1为true的那些位置"
% x(L1) 表示"x数组中L1为true的那些元素"
% y(L1) = x(L1) 表示"把右边选中的元素赋值给左边选中的位置"

% 调用：x = -2:0.01:2; y2 = exm030303_2(x);

% 验证两种方法的一致性
y1 = exm030303_1(x);
y2 = exm030303_2(x);
max_error = max(abs(y1(:) - y2(:)))
max_error = 0              % 结果完全相同！
```

## 十一、矩阵软阈值算法[^1]

```matlab
% 问题：将矩阵A中所有元素转换为阈值处理后的元素
% b_ij = T₁(|a_ij|) = {  0,                         |a_ij| <= 1
%                      {  sign(a_ij)·(|a_ij| - 1), |a_ij| > 1

A = [1, -2, 0, 1; -1, 2, -3, 4; 1, -3, 5, 0];

% 低效方法：两层for循环

for i = 1:3
    for j = 1:4
        if abs(A(i,j)) <= 1
            B(i,j) = 0;
        else
            B(i,j) = sign(A(i,j)) * (abs(A(i,j)) - 1);
        end
    end
end

% 高效方法：数组化运算（一行代码！）
B = max(abs(A) - 1, 0) .* sign(A);
% 原理：
% abs(A) - 1 计算所有元素的"超额值"
% max(..., 0) 将负值置为0（对应|a_ij| <= 1的情况）
% .* sign(A) 乘以符号，恢复正负性
```

```matlab
% 🔑 考试陷阱：
% - 逻辑索引返回列向量，不保持原形状！
% - 单下标按列优先顺序
% - A(:)把矩阵展开为列向量
```


#### 模板3：逻辑数组应用（⭐⭐⭐ 必考）

```matlab
% 任务：将矩阵中所有负数替换为0

A = [1, -2, 3; -4, 5, -6; 7, -8, 9];

% ❌ 低效方法：双重循环
B = A;
for i = 1:3
    for j = 1:3
        if B(i,j) < 0
            B(i,j) = 0;
        end
    end
end

% ✅ 高效方法：逻辑索引（一行代码）⭐⭐⭐
C = A;
C(C < 0) = 0;                   % 所有<0的元素赋值为0
% C = [1 0 3; 0 5 0; 7 0 9]

% 更多逻辑索引应用
A(A > 5) = NaN;                 % >5的设为NaN
A(A == 0) = [];                 % 删除所有0（变为列向量）

% 🔑 考试模板：
% 1. 创建逻辑矩阵：L = (条件表达式)
% 2. 应用：A(L) = 新值
% 3. 提取：selected = A(L)

% 复杂条件
L = (A > 0) & (A < 10);         % 与运算
A_filtered = A(L);

L2 = (A < 0) | (A > 100);       % 或运算
outliers = A(L2);

% 🤔 为什么结果是列向量？详细解释：
% 原矩阵：A = [1, -2, 3; -4, 5, -6; 7, -8, 9]

% 步骤1：理解逻辑矩阵L的结构
L = (A > 0) & (A < 10)
% L = [true, false, true; false, true, false; true, false, true]
%     位置：(1,1)=T, (1,2)=F, (1,3)=T
%           (2,1)=F, (2,2)=T, (2,3)=F  
%           (3,1)=T, (3,2)=F, (3,3)=T

% 步骤2：MATLAB按列优先顺序提取元素
% MATLAB内部存储矩阵是按列存储的：
% A的内部顺序：[1, -4, 7, -2, 5, -8, 3, -6, 9]
%              第1列   第2列    第3列

% 步骤3：按列优先顺序检查逻辑条件
% L=true的位置按列优先顺序：
% (1,1)→1, (3,1)→7, (2,2)→5, (1,3)→3, (3,3)→9
% 所以 A_filtered = [1; 7; 5; 3; 9] （列向量）

% 🔑 关键规则：逻辑索引 A(L) 总是返回列向量！
% 这是MATLAB的设计规范，无论原矩阵是什么形状

% 验证L2的情况：
L2 = (A < 0) | (A > 100)
% L2 = [false, true, false; true, false, true; false, true, false]
% L2=true的位置按列优先：(2,1)→-4, (1,2)→-2, (3,2)→-8, (2,3)→-6
% 所以 outliers = [-4; -2; -8; -6] （列向量）

% 如果想保持原矩阵形状，需要其他方法：
A_copy = A;
A_copy(~L) = NaN;              % 不满足条件的设为NaN，保持矩阵形状
% A_copy = [1, NaN, 3; NaN, 5, NaN; 7, NaN, 9]
```


#### 模板4：向量化编程标准套路（⭐⭐⭐）

```matlab
% 任务：计算分段函数（向量化）
% f(x) = { x,        x≤-1
%        { x^3·cos(2πx), -1<x≤1
%        { e^(-x+1),   x>1

function y = piecewise_vectorized(x)
    % ✅ 高效向量化实现
    
    % 步骤1：创建逻辑数组标记各段
    L1 = (x <= -1);
    L2 = (x > -1) & (x <= 1);
    L3 = (x > 1);
    
    % 步骤2：预分配结果数组
    y = zeros(size(x));
    
    % 步骤3：分段赋值（无循环！）
    y(L1) = x(L1);
    y(L2) = x(L2).^3 .* cos(2*pi*x(L2));
    y(L3) = exp(-x(L3) + 1);
    
    % 🔑 要点：
    % - 必须用数组运算符（.*、.^）
    % - 每段独立计算，避免相互干扰
end

% 测试
x = -2:0.01:2;
y = piecewise_vectorized(x);
plot(x, y, 'LineWidth', 2)
xlabel('x'), ylabel('y')
title('Piecewise Function (Vectorized)')
grid on
```


#### 模板5：矩阵软阈值（⭐⭐ 常考算法）

```matlab
% 任务：软阈值算子 T_λ(a) = sign(a)·max(|a|-λ, 0)

A = randn(5, 5);                % 随机矩阵
lambda = 0.5;                   % 阈值

% ❌ 低效：双重循环+if
B_slow = zeros(size(A));
for i = 1:5
    for j = 1:5
        if abs(A(i,j)) > lambda
            B_slow(i,j) = sign(A(i,j)) * (abs(A(i,j)) - lambda);
        end
    end
end

% ✅ 高效：向量化（一行代码）⭐⭐⭐
B_fast = sign(A) .* max(abs(A) - lambda, 0);

% 验证
fprintf('两种方法差异：%.2e\n', max(abs(B_slow(:) - B_fast(:))));

% 🔑 算法理解：
% - abs(A) - lambda：计算"超额值"
% - max(..., 0)：负值置0
% - sign(A)：恢复符号
% - 三者相乘得软阈值结果
```

#### 错误1：单下标和双下标混淆

```matlab
A = reshape(1:6, 2, 3);         % 2×3矩阵
% A = [1 3 5]
%     [2 4 6]

% ❌ 错误理解
% 以为A(5)是"第5个"→应该是什么？

% ✅ 正确理解：列优先
% A(1)=1, A(2)=2, A(3)=3, A(4)=4, A(5)=5, A(6)=6
A(5)                            % = 5（第3列第1行）

% 转换
[i, j] = ind2sub([2, 3], 5);    % i=1, j=3 ✓
k = sub2ind([2, 3], 1, 3);      % k=5 ✓
```

### 快速记忆技巧

**1. 数组生成**：
- **冒号**：知道步长用`:`
- **linspace**：知道点数用`linspace`⭐
- **logspace**：对数尺度用`logspace`

**2. 矩阵寻址**：
- **单冒号**：`A(:)`展开为列向量

**3. 向量化口诀**：
- **能用矩阵，不用循环**
- **数组运算，别忘加点**
- **逻辑索引，一步到位**
