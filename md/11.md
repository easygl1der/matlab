## 一、数字信号基础概念

### 1.1 信号的数字化过程[^1]

**三个关键步骤**：

1. **抽样（采样）**：连续信号在等时间间隔的取值
2. **量化**：连续值转为离散值
3. **编码**：离散值转为二进制代码

**数字信号的特点**：

- **离散性**：自变量和函数值都是离散定义的
- **采样率**：每秒采样的次数（如44100Hz）
- 数字信号的定义形式：$\{x_i, i=1,2,\ldots,n\}$


### 1.2 数字信号的分类[^1]

1. **波形信号**：光滑周期函数，可通过拟合/插值恢复
2. **声音信号**：包含多种频段，通过傅里叶变换频率分析
3. **分片光滑/分片常函数**：有跳跃点
4. **随机函数**：符合特定分布

## 二、噪声与信噪比

### 2.1 理想信号与含噪信号[^1]

**例**：真实信号 $f(x) = \sin x$，$x \in [0, 2\pi]$

- **采样**：均匀100等分采样
- **加性高斯噪声**：$\epsilon \sim N(0, \sigma^2)$，$\sigma = 0.1$
- **含噪信号**：$y(x) = f(x) + \epsilon$


### 2.2 噪声强度的度量[^1]

#### 1. 平方和误差（SE）

$SE = \sum (y_i - f(x_i))^2$
越小越好

#### 2. 平均平方误差（MSE）

$MSE = \frac{SE}{n}$
与原数据量纲一致

#### 3. 信噪比（SNR - Signal-to-Noise Ratio）[^1]

$SNR = 10 \cdot \log_{10}\left(\frac{\sum f^2(x_i)}{\sum (y - f(x))^2}\right)$

**性质**：

- 与真实信号特性有关
- SNR越高，信号质量越高
- SNR > 10：噪声强度不足信号的 $\frac{1}{\sqrt{10}}$

**MATLAB函数**：

```matlab
snr_value = snr(s, n)  % s为信号，n为误差
```

**例**：原始噪声SNR = 15.6547

## 三、高斯噪声滤波方法

### 3.1 五点均值滤波（smooth函数）[^1]

```matlab
% 基本用法
f1 = smooth(y)  % 五点均值滤波（默认）
f2 = smooth(f1) % 二次均值滤波
f3 = smooth(f2) % 三次均值滤波

% SNR变化过程
% 原始：SNR = 15.6547
% f1（一次）：SNR = 22.3808
% f2（二次）：SNR = 24.0649
% f3（三次）：SNR = 24.8336
```

**原理**：

- 高斯噪声是零均值
- 噪声幅度小于信号强度
- 局部均值消除噪声


### 3.2 卷积滤波（conv函数）[^1]

#### 1. 五点均值滤波

```matlab
f1 = conv(y, ones(1,5)*1/5, 'same')
% SNR = 22.3660
```


#### 2. 五点加权均值滤波

```matlab
f2 = conv(y, [1/16 1/4 3/8 1/4 1/16], 'same')
% SNR = 20.9296（本例效果较差）
```


#### 3. 九点均值滤波

```matlab
f3 = conv(y, ones(1,9)*1/9, 'same')
% SNR = 24.3451
% 滤波范围更大，效果更好
% 但范围过大会导致过度平滑、失真
```

**'same'参数**：保持输出长度与输入相同

## 四、傅里叶变换

### 4.1 连续傅里叶变换（复习）[^1]

#### 定义

$F(\omega) = \int_{-\infty}^{+\infty} f(t) e^{-j\omega t} dt$

其中 $j$ 为虚数单位，$e^{-j\omega t} = \cos(\omega t) - j \sin(\omega t)$

#### 逆变换

$f(t) = \frac{1}{2\pi} \int_{-\infty}^{+\infty} F(\omega) e^{j\omega t} d\omega$

#### Parseval等式（能量守恒）

$\int_{-\infty}^{+\infty} |f(t)|^2 dt = \frac{1}{2\pi} \int_{-\infty}^{+\infty} |F(\omega)|^2 d\omega$

#### 时频分析意义

- $t$：时间
- $\omega$：频率
- $F(\omega)$ 大的地方表示该频率分量强


#### MATLAB函数

```matlab
F = fourier(f, t, w)      % 时域→频域（f_t为时间函数，t变量，w频率变量）
f = ifourier(F, w, t)     % 频域→时域
```


### 4.2 离散傅里叶变换（DFT）[^1]

#### 定义（1-indexed）

$Y_k = \sum_{i=1}^{n} X_i e^{-2\pi j \frac{(i-1)(k-1)}{n}}, \quad 1 \leq k \leq n$

**特殊情况**：当 $k=1$ 时，$Y_1 = \sum_{i=1}^{n} X_i$（低频系数，表达函数总和）

其余系数为**高频傅里叶系数**，刻画函数变化情况

#### 逆变换

$X_i = \frac{1}{n} \sum_{k=1}^{n} Y_k e^{2\pi j \frac{(i-1)(k-1)}{n}}, \quad 1 \leq i \leq n$

**互逆性**：完全可逆

#### MATLAB函数

```matlab
Y = fft(X)      % 离散傅里叶变换
X = ifft(Y)     % 离散傅里叶逆变换
```


#### DFT的工作原理[^1]

- 将有限长度序列 $X(i)$ 假想为以 $n$ 为周期的无限周期序列
- 利用有限个特定频率采样完成频率表示
- 从时域转到频域


### 4.3 利用DFT进行高斯噪声去噪[^1]

**算法**：

1. 对含噪信号做DFT
2. 仅保留绝对值 $> \lambda$ 的系数（硬阈值）
3. 对其他系数设为0
4. 用IDFT恢复

**代码示例**：[^1]

```matlab
clear, close all, rng default
x = 0:pi/50:2*pi;
f = sin(x);
y = sin(x) + randn(size(x))*0.1;

% 绘制频域信息
clf, hold on
plot(abs(fft(f)), 'LineWidth', 2)
plot(abs(fft(y)), 'r:', 'LineWidth', 2)
legend('真实信号傅里叶变换', '含噪信号傅里叶变换', 'Location', 'best')
hold off

% DFT去噪
Ff = fft(y);
Ff(abs(Ff) < 5) = 0.0;      % 硬阈值：去掉<5的系数
f1 = ifft(Ff);              % 逆变换

% 计算SNR
snr(f, f1-f)
% 结果：SNR = 30.4972
```

**观察**：

- 真实信号傅里叶变换仅在 $k=2$ 和 $k=n$ 处有大系数
- 含噪信号在中间部分有额外的非零系数
- 硬阈值去掉这些中间系数，恢复效果好

## 五、硬阈值与软阈值算法

### 5.1 硬阈值算子[^1]

$T_\lambda(y) = \begin{cases} y, & |y| \geq \lambda \\ 0, & |y| < \lambda \end{cases}$

**性质**：绝对值 $\geq \lambda$ 的元素保持不变

**优化问题**：
$\min_x \frac{1}{2}\|x - y\|_2^2 + \frac{\lambda^2}{2}\|\hat{x}\|_0$

其中 $\hat{x}$ 是 $x$ 的离散傅里叶变换，$\|\cdot\|_0$ 是非零元素个数

### 5.2 软阈值算子[^1]

$\mathcal{T}_\lambda(y) = \text{sign}(y) \cdot \max(|y| - \lambda, 0)$

**特性**：

- 模大于 $\lambda$ 的系数：幅角不变，模减 $\lambda$
- 模 $\leq \lambda$ 的系数：设为0

**优化问题**：
$\min_x \frac{1}{2}\|x - y\|_2^2 + \lambda\|\hat{x}\|_1$

### 5.3 软阈值与硬阈值的对比[^1]

**代码示例**：[^1]

```matlab
clear, close all, rng default
x = 0:pi/50:2*pi;
f = sin(x);

% 构造分片光滑函数（添加阶梯）
for i = 1:5
    f(i*20-19:i*20) = f(i*20-19:i*20) + (5-i);
end

y = f + randn(size(x))*0.1;

% 软阈值算法
Ff = fft(y);
Ff = sign(Ff) .* max(abs(Ff) - 5, 0);  % 软阈值
f1 = ifft(Ff);

% 结果对比
snr(f, f1-f)
% 软阈值SNR = 18.3973
% 硬阈值SNR = 14.2758
% 五点均值SNR = 19.5122
```

**观察**：

- 软阈值优于硬阈值（避免不连续）
- 但仍不如五点均值（分片函数傅里叶系数稀疏度不足）

## 六、数字信号去噪综述[^1]

### 6.1 去噪方法的选择[^1]

**关键因素**：

1. **噪声类型**
2. **噪声强度**
3. **目标函数特性**（关键！）

**不存在一劳永逸的通用方法**

### 6.2 常见噪声类型[^1]

#### 1. 高斯噪声

- 加性：$y = f(x) + \epsilon$，$\epsilon \sim N(0, \sigma^2)$
- 处理方法：均值滤波、傅里叶硬阈值


#### 2. 冲击噪声（脉冲噪声）

- 随机采样值出现与真实值无关的错误
- 处理方法：**中值滤波**（median filter）


#### 3. 泊松噪声（非加性）

- 基于参数 $\lambda = k \cdot \text{真实值}$ 的泊松分布
- 带噪信号该点值为 $\frac{Y}{k}$，其中 $Y \sim \text{Poisson}(\lambda)$
- $k$ 越小，噪声影响越大


### 6.3 方法的适用性[^1]

| 去噪方法 | 最佳用途 | 说明 |
| :-- | :-- | :-- |
| **均值滤波** | 高斯噪声 | 简单有效，适合光滑信号 |
| **傅里叶硬阈值** | 三角函数 | 对周期函数效果极佳 |
| **傅里叶软阈值** | 周期信号 | 避免不连续 |
| **中值滤波** | 脉冲噪声 | 对孤立尖峰有效 |
| **分片光滑** | 有跳跃点 | 需要适应信号特征 |

**核心建议**：根据目标函数特性选择合适的变换及正则化方法

## 八、声音的物理特性

### 8.1 响度与振幅[^1]

**响度**（loudness）：

- 振动的**振幅**
- 表示某时段的振动强度
- 复合振动中某声部的强度


### 8.2 音色与波形[^1]

**音色**（timbre）：

- 由**波形**决定
- 不同乐器波形不同→不同音色

**例**：

- 音叉：纯正三角波
- 钢琴、长笛：复杂波形（多次谐振）
- 合成三角波：声音单纯发涩（缺乏谐振）

**应用**：

- 提取乐器音色特征进行合成
- 电子琴、MIDI技术基于此原理
- 歌手人声仍难以完美复制和迁移


### 8.3 音调与频率[^1]

**关系**：

- 频率高↔音调高
- 每升**一个八度**→频率×2
- 每升**一个半音**→频率×$2^{1/12}$

**标准音高**（中央C及各音）：


| 音名 | do | re | mi | fa | so | la(标准) | ti | 高do |
| :-- | :-- | :-- | :-- | :-- | :-- | :-- | :-- | :-- |
| 频率(Hz) | 261.6 | 293.6 | 329.6 | 349.2 | 392.0 | **440.0** | 493.8 | 523.2 |

**MIDI音高编码**：每个八度12个半音（对数等距）

### 8.4 变频播放[^1]

```matlab
% 不改变波形，改变采样率
sound(ys, Fs/sqrt(2))
% 效果：速度减慢1/√2倍，音调降低半个八度（6个半音）
% 从F#调变成C调

% 原理：采样率决定播放速度和音调
```

## 九、声音频率分析

### 9.1 全局傅里叶变换[^1]

```matlab
plot(abs(fft(ys)))
% 难以看出细微的音调变化
```

**局限性**：

- 只能看全局频率范围
- 音调通常在0.5秒甚至更短时间内变化


### 9.2 短时傅里叶变换（STFT）[^1]

**原理**：

1. 设置窗口大小（如4410个样本 = 0.1秒）
2. 对每个时间点的小窗口做FFT
3. 记录最大模所在的频率

**代码**：[^1]

```matlab
ub = length(ys) - 4410;
height = zeros(ub, 1);

% 注：为了效率可改为 1:100:ub
for i = 1:ub
    temp = ys(i:i+4410);        % 截取4410个样本
    tempf = fft(temp);
    [maxv, maxp] = max(abs(tempf));  % 找最大模位置
    height(i) = maxp;           % 记录频率位置
end

% 绘制音高变化
plot(1:ub, height)
```

**窗口大小选择**：

- **过大**：混入多个时间段的音调
- **过小**：不包含足够主旋律周期


### 9.3 音高分析的应用[^1]

**例**：黄晓明唱歌音频分析

- 可检测到"破音"现象（频率断层）

**局限性**：[^1]

- 多声部音乐：关心的声部可能不是最大振幅
- 识别准确度下降或失效
- 需要更高级的信号处理技术

## 十、音符与音乐合成

### 10.1 基本音符构造[^1]

**特点**：

- 从无振动→振幅渐变（先强后弱）→逐渐停止
- 频率与音调相关
- 包含特定数量的振动周期

**代码示例**（1秒的A调音符440Hz）：[^1]

```matlab
fs = 44100;
t = 0:44100;

% 包络线函数：先强后弱（半周期三角波）
mod = sin(t/44100*pi);

% 音符：包络×载波（440个周期）
note = mod .* cos(t/44100*pi*2*440.0);

plot(t, note)
```

**结构**：

```
envelope × carrier wave = final sound
```

其中 envelope 控制响度变化，carrier wave 决定音调

### 10.2 音乐设计原理[^1]

**步骤**：

1. **音符库**：预制各种长度、音高的音符向量
2. **拼接**：左右顺序连接（播放一个接一个）
3. **叠加**：向量相加（听到多个声音混合）
4. **归一化**：确保绝对值 $\leq 1$（防止失真）

**播放**：

```matlab
sound(music_signal, Fs)
```


### 10.3 转音效果（变频音符）[^1]

**本质**：连续变化频率

**模拟消防车鸣笛**：

```matlab
% 频率从低→高→低
% 以对数速度变化，如1215个周期跨越一个八度
% 每个频率提供一个周期的振动
% 串联→加包络线
```

**实现**：

- 预设频率渐变速度
- 每个频率贡献一个周期
- 不同频率波串联

| **硬阈值** | `Ff(abs(Ff)<λ)=0` | 去掉小于λ的系数 |
| **软阈值** | `Ff = sign(Ff).*max(abs(Ff)-λ,0)` | 缩小系数模 |
| **音频信息** | `audioinfo(filename)` | 读取文件信息 |
| **读取音频** | `[y, Fs]=audioread(file)` | y为数据，Fs为采样率 |
| **播放声音** | `sound(y, Fs)` | 按采样率Fs播放 |
| **停止播放** | `clear sound` | 停止声音 |
| **存储音频** | `audiowrite(file, y, Fs)` | 存为指定格式 |
| **截取信号** | `ys = y(1:11*Fs, 1)` | 截取前11秒左声道 |
| **变速播放** | `sound(ys, Fs/k)` | 采样率÷k，速度和音调改变 |
| **短时FFT** | `fft(y(i:i+N))` | 窗口大小N |

## 十四、关键概念速记

### 信号处理流程

```
原始信号 → [添加噪声] → 含噪信号 → [选择滤波方法]
    ↓
  去噪 → 恢复信号 → [评估SNR] → 对比原信号
```


### 去噪方法选择树

```
高斯噪声？
├─ 是 → 信号光滑？
│      ├─ 是 → 均值滤波 ✓
│      └─ 否 → 傅里叶阈值
└─ 否 → 噪声类型？
       ├─ 脉冲 → 中值滤波
       ├─ 泊松 → 特殊处理
       └─ 未知 → 综合方法
```


### 音乐三要素对应物理量

```
响度 (Loudness) ← 振幅 (Amplitude)
音调 (Pitch)    ← 频率 (Frequency)
音色 (Timbre)   ← 波形 (Waveform)
```


### STFT窗口选择权衡

```
窗口过大 → 时间分辨率差，频率分辨率好 → 混入多音高
窗口过小 → 时间分辨率好，频率分辨率差 → 识别不准
最优    → 0.05~0.1秒的窗口
```

### 核心代码模板

#### 模板1：傅里叶去噪标准流程（⭐⭐⭐ 必考）

```matlab
% 任务：用FFT去除高斯噪声

% 步骤1：生成含噪信号
rng default
x = 0:pi/50:2*pi;               % 采样点
f = sin(x);                     % 原始信号
noise = 0.1 * randn(size(x));   % 高斯噪声
y = f + noise;                  % 含噪信号

% 步骤2：傅里叶变换
F_clean = fft(f);               % 原信号频谱
F_noisy = fft(y);               % 含噪信号频谱

% 步骤3：观察频谱⭐⭐
figure
subplot(2,1,1)
plot(abs(F_clean), 'b', 'LineWidth', 2)
title('原信号频谱')
ylabel('|F(k)|')

subplot(2,1,2)
plot(abs(F_noisy), 'r', 'LineWidth', 2)
title('含噪信号频谱')
xlabel('频率索引k'), ylabel('|F(k)|')

% 步骤4：硬阈值去噪⭐⭐⭐
lambda = 5;                     % 阈值
F_denoised = F_noisy;
F_denoised(abs(F_denoised) < lambda) = 0;

% 步骤5：逆变换恢复信号
f_recovered = ifft(F_denoised);
f_recovered = real(f_recovered); % 取实部

% 步骤6：评估（信噪比）
snr_original = snr(f, noise);
snr_recovered = snr(f, f_recovered - f);
fprintf('原始SNR：%.2f dB\n', snr_original);
fprintf('去噪后SNR：%.2f dB\n', snr_recovered);

% 步骤7：可视化对比
figure
plot(x, f, 'b', 'LineWidth', 2)
hold on
plot(x, y, 'r:', 'LineWidth', 1)
plot(x, f_recovered, 'g--', 'LineWidth', 2)
legend('原信号', '含噪信号', '去噪信号')
xlabel('x'), ylabel('y')
title('FFT硬阈值去噪')
grid on

% 🔑 考试要点：
% - fft转频域，观察主要频率成分
% - 噪声在频域分散，信号集中
% - 硬阈值：小于λ的置0
% - ifft恢复时域信号
% - SNR评估去噪效果
```


#### 模板2：软阈值vs硬阈值（⭐⭐⭐）

```matlab
% 对比两种阈值方法

F = fft(y);                     % 含噪信号频谱
lambda = 5;

% 硬阈值⭐⭐⭐
F_hard = F;
F_hard(abs(F_hard) < lambda) = 0;
% T_λ(y) = { y,  |y|≥λ
%          { 0,  |y|<λ

% 软阈值⭐⭐⭐
F_soft = sign(F) .* max(abs(F) - lambda, 0);
% T_λ(y) = sign(y)·max(|y|-λ, 0)

% 对比
f_hard = real(ifft(F_hard));
f_soft = real(ifft(F_soft));

snr_hard = snr(f, f_hard - f);
snr_soft = snr(f, f_soft - f);

fprintf('硬阈值SNR：%.2f dB\n', snr_hard);
fprintf('软阈值SNR：%.2f dB\n', snr_soft);

% 🔑 区别：
% - 硬阈值：保持不变或置0（不连续）
% - 软阈值：保持或缩小（连续）
% - 软阈值通常更平滑
% - 对分片光滑信号，软阈值更优
```


#### 模板3：卷积滤波（⭐⭐）

```matlab
% 使用卷积实现各种滤波

y = sin(x) + 0.1*randn(size(x)); % 含噪信号

% 滤波器1：五点均值
h1 = ones(1, 5) / 5;
y1 = conv(y, h1, 'same');       % 'same'保持长度
snr1 = snr(f, y1 - f);

% 滤波器2：加权均值
h2 = [1/16, 1/4, 3/8, 1/4, 1/16];
y2 = conv(y, h2, 'same');
snr2 = snr(f, y2 - f);

% 滤波器3：九点均值
h3 = ones(1, 9) / 9;
y3 = conv(y, h3, 'same');
snr3 = snr(f, y3 - f);

fprintf('五点均值SNR：%.2f\n', snr1);
fprintf('加权均值SNR：%.2f\n', snr2);
fprintf('九点均值SNR：%.2f\n', snr3);

% 🔑 考试要点：
% - conv(y, h, 'same')保持长度
% - 滤波器h的和应为1（保持幅度）
% - 窗口越大，越平滑，但可能过度
% - 等效于smooth(y)（五点）
```

### 去噪方法选择指南

### 考前必记公式

**1. 离散傅里叶变换（DFT）**：

$$Y_k = \sum_{n=1}^{N} x_n e^{-2\pi i(n-1)(k-1)/N}$$

**2. 信噪比（SNR）**：

$$SNR = 10 \log_{10} \frac{\sum f^2(x_i)}{\sum (y_i - f(x_i))^2}$$

**3. 音调与频率关系**：

- 升一个八度：频率×2
- 升一个半音：频率×$2^{1/12}$
