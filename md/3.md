## 三、符号表达式的化简（simplify）[^1]

### 基本用法

```matlab
clear all
syms x

% 复杂的三角函数表达式
f1 = ((exp(-x*i)*i)/2 - (exp(x*i)*i)/2) / (exp(-x*i)/2 + exp(x*i)/2)
f1 = ((exp(-x*1i)*1i)/2 - (exp(x*1i)*1i)/2)/(exp(-x*1i)/2 + exp(x*1i)/2)

% 普通方法化简1步
S11 = simplify(f1)
S11 = -(exp(x*2i)*1i - 1i)/(exp(x*2i) + 1)

% 不同方法化简40步（使用'Steps'参数）
S12 = simplify(f1, 'Steps', 40)
S12 = tan(x)                  % 最终化简为tan(x)！

% 验证：使用循环重复应用simplify
w = f1;
for k = 1:40
    w = simplify(w);
end
S13 = w
S13 = -(exp(x*2i)*1i - 1i)/(exp(x*2i) + 1)  % 普通方法无法继续化简
```

### 使用特殊选项参数化简[^1]

#### 'Criterion'参数：控制化简的标准

```matlab
% 使用'Criterion'参数控制化简结果的形式
f2 = sym(1i)^(1i+1)
f2 = 1i^(1 + 1i)

% 标准化简（化为指数形式）
S21 = simplify(f2, 'Steps', 100)
S21 = (-1)^(1/2 + 1i/2)       % 结果包含虚数在括号内层

% 使用'preferReal'标准（尽可能避免括号内层出现虚数）
S22 = simplify(f2, 'Steps', 100, 'Criterion', 'preferReal')
S22 = exp(-pi/2)*1i           % 更简洁的实数指数形式
```

#### 'IgnoreAnalyticConstraints'参数[^1]

```matlab
syms x

% 定义复杂的立方根表达式
f = (1/x^3 + 6/x^2 + 12/x + 8)^(1/3)
f = (12/x + 6/x^2 + 1/x^3 + 8)^(1/3)

% 标准化简（保守）
g1 = simplify(f)
g1 = ((2*x + 1)^3/x^3)^(1/3)  % 只能化到这一步

% 忽略复数域中可能的重根问题（激进化简）
g2 = simplify(f, 'Steps', 10, 'IgnoreAnalyticConstraints', true)
g2 = 1/x + 2                  % 最终化简结果

% 注意：'IgnoreAnalyticConstraints'选项会忽略定义域的某些约束
% 使用时要谨慎！
```

### 化简的其他方式

```matlab
syms x t

% 定义表达式

% collect函数：合并同类项

% pretty函数：按照书写格式美化输出
pretty(f1)
% 输出：
% 3            2

%  (输出为手写格式的数学表达式)

% expand函数：展开表达式
f_expanded = expand(f)

```

## 四、通用置换命令（subs）[^1]

### 基本用法

```matlab
clear
syms a b y x

% 定义符号表达式
f1 = a*sin(x) + b
f1 = b + a*sin(x)

% 方式1：替换子表达式
f2 = subs(f1, sin(x), log(y))  % 整体代换sin(x)为log(y)
f2 = b + a*log(y)
class(f2)
ans = sym                      % 结果仍为sym型

% 方式2：替换为具体数值
f3 = subs(f1, a, sym(3.11))
f3 = b + (311*sin(x))/100
class(f3)
ans = sym

% 方式3：用向量代换变量
f4 = subs(f1, x, [0, pi/2, pi])  % 将x替换为三个值
f4 = [ b, a + b, b]
class(f4)
ans = sym                       % 结果为向量
```

### 多参数替换[^1]


## 五、符号数值转换[^1]

### 符号型到双精度的转换

```matlab

a = double(A)              % 转换为双精度
```

### 精度控制函数[^1]

`vpa(x)` 将符号输入 `x` 转换为高精度浮点数，精度由 `digits` 函数控制，默认 32 位，比 `double`（约 15 位）更精确。

```matlab
% 1. digits函数：设置和查看vpa的精度
digits                     % 查看当前精度
% Digits = 32             （默认32位有效数字）

digits(n)                  % 设置精度为n位有效数字
digits(48)                 % 改为48位精度

% 2. vpa函数：按指定精度显示
sa = sym(1/3) + sqrt(sym(2))
sa = 2^(1/2) + 1/3

% 使用vpa显示不同精度
sa32 = vpa(sa)             % 按当前digits(32)显示
sa32 = 1.747546895706428382135022057543

digits(48)                 % 改为48位精度
sa48 = vpa(sa)             % 再用vpa显示
sa48 = 1.74754689570642838213502205754303141190300520871

% 也可以直接指定精度
sa5 = vpa(sa, 5)           % 只显示5位有效数字
sa5 = 1.7475

% 3. 混合运算的精度演示
format long
a_numerical = 1/3 + sqrt(2)      % 数值计算
a_numerical = 1.747546895706428

% 符号与数值混合运算
sa_plus_a = vpa(sa + a_numerical, 20)
sa_plus_a = 3.4950937914128567869

sa_minus_a = vpa(sa - a_numerical, 20)
sa_minus_a = -0.000000000000000022658064826339973669
% 差值显示了double精度的限制

% 4. 转为双精度（允许误差）
sad = double(sa)
sad = 1.747546895706428          % 与数值计算结果相同
```

## 六、阿基米德螺线长度的计算与图示[^1]

### 微分与积分计算

```matlab
% 例：求阿基米德螺线 r = a·θ 的曲线长度函数
% 曲线长度公式：L(φ) = ∫₀^φ √[(dr/dθ)² + r²] dθ

syms a r th p                  % a为参数，r为半径，th为角度，p为积分上限

% 极坐标参数化
r = a*th;                      % r = a·θ
x = r*cos(th);                 % x = r·cos(θ)
y = r*sin(th);                 % y = r·sin(θ)

% 计算导数
dx_dth = diff(x, th)           % ∂x/∂θ
dy_dth = diff(y, th)           % ∂y/∂θ

% 计算被积函数
dL = simplify(sqrt(dx_dth^2 + dy_dth^2), ...
              'IgnoreAnalyticConstraints', true);

% 计算曲线长度积分
L = int(dL, th, 0, p)
L = (a*(asinh(p) + p*(p^2 + 1)^(1/2)))/2
% 曲线长度函数为：L(φ) = (a/2)[sinh⁻¹(φ) + φ√(φ²+1)]
```

### 曲线长度的数值计算

```matlab
% 计算特定点的长度
L2pi = subs(L, [a, p], [1, 2*pi])  % a=1, φ=2π
L2pi = asinh(2*pi)/2 + pi*(4*pi^2 + 1)^(1/2)

% 转为数值
L2pid = vpa(L2pi)
L2pid = 21.256294148209098800702512272566
% 在[0, 2π]范围内，螺线长度约为21.26
```

### 螺线的参数绘图[^1]

```matlab

hold on

% 绘制原始螺线

% 添加图例

```

```matlab
disp('微分方程的解：')
```

## 八、傅里叶变换（Fourier Transform）[^1]

### 基本概念

**傅里叶变换定义**：
$$
F(\omega) = \int_{-\infty}^{+\infty} f(t)e^{-j\omega t}dt
$$


**傅里叶逆变换定义**：
$$

f(t) = \frac{1}{2\pi}\int_{-\infty}^{+\infty} F(\omega)e^{j\omega t}d\omega

$$
其中$e^{-j\omega t} = \cos(\omega t) - j \sin(\omega t)$

**帕萨瓦尔等式**：
$$

\int_{-\infty}^{+\infty}|f(t)|^2 dt = \frac{1}{2\pi}\int_{-\infty}^{+\infty}|F(\omega)|^2 d\omega

$$
### 傅里叶变换的计算[^1]

```matlab
syms t w

% Heaviside单位阶跃函数
ut = heaviside(t);

% 计算傅里叶变换
UT = fourier(ut)
UT = pi*dirac(w) - 1i/w
% 结果包含Dirac δ函数（冲激函数）

% 计算傅里叶逆变换
Ut = ifourier(UT, w, t)
Ut = (pi + pi*sign(t))/(2*pi)

% 验证逆变换的正确性
Err = simplify(Ut - ut)
Err = 0                        % 误差为0，验证成功！
```

### 矩形脉冲的傅里叶变换[^1]

```matlab
% 例：矩形脉冲 y(t) = {A, |t| < τ/2; 0, |t| > τ/2 的Fourier变换

syms A t w tao

% 利用两个Heaviside函数定义矩形脉冲
yt = A*(heaviside(t + tao/2) - heaviside(t - tao/2));

% 计算傅里叶变换
Yw = fourier(yt, t, w)
Yw = A*((cos((tao*w)/2)*i + sin((tao*w)/2))/w - ...)

% 化简
Yws = simplify(Yw)
Yws = (2*A*sin((tao*w)/2))/w  % 辛格函数sinc

% 计算傅里叶逆变换验证
Yt = ifourier(Yws, w, t)
Yts = simplify(Yt)
```

## 十、拉普拉斯变换（Laplace Transform）[^1]

### 基本定义

**拉普拉斯变换定义**：
$$

F(s) = \int_{0}^{+\infty} f(t)e^{-st}dt

$$
其中s为复数，通常表示为$s = \sigma + j\omega$

**拉普拉斯逆变换定义**：
$$

f(t) = \frac{1}{2\pi j}\int_{c-j\infty}^{c+j\infty} F(s)e^{st}ds

$$
### 拉普拉斯变换的计算[^1]

```matlab
syms t s a b

% 计算e^(-at)·sin(bt)的拉普拉斯变换
f1 = exp(-a*t)*sin(b*t);
F1 = laplace(f1, t, s)
F1 = b/((a + s)^2 + b^2)      % 结果为有理函数

% 处理Heaviside函数
syms a positive                % 声明a为正实数符号变量

f2 = heaviside(t - a);
F2 = laplace(f2, t, s)
F2 = exp(-a*s)/s              % 时滞性质

% 处理Dirac冲激函数
f3 = dirac(t - b);            % b仍为符号变量
F3 = laplace(f3, t, s)
F3 = piecewise([b < 0, 0], [0 <= b, exp(-b*s)])
% 结果根据b的符号返回不同值

% 当b为正实数时
f4 = dirac(t - a);            % a已声明为正实数
F4 = laplace(f4, t, s)
F4 = exp(-a*s)                % 简洁的结果
```


### 拉普拉斯逆变换[^1]

```matlab
% 逆变换验证
syms t s a
f5 = dirac(t - a);
F5 = laplace(f5, t, s)
F5 = exp(-a*s)

% 计算逆变换
ft_F5 = ilaplace(F5, s, t)
ft_F5 = dirac(a - t)          % 恢复到δ(t-a)的形式

% 处理幂函数的拉普拉斯变换
% ℒ[t^n] = n!/(s^(n+1))

n = sym('n', 'positive');      % n为正实数
F6 = laplace(t^n, t, s)
F6 = gamma(n + 1)/s^(n + 1)
% gamma(n+1) = n! 当n为正整数时

% 例：F(s) = e^(-a/s)的逆变换
syms a z n
GZ = exp(-a/z);
gn = iztrans(GZ, z, n)
gn = (-a)^n/factorial(n)      % 结果为幂级数的系数
```


## 十一、拉普拉斯变换的卷积公式[^1]

### 卷积的定义与计算

**非负区间卷积定义**：
$$

(u * h)(t) = \int_{0}^{t} u(\tau)h(t-\tau)d\tau

$$
**卷积的拉普拉斯变换性质**：
$$

\mathcal{L}\{u * h\} = \mathcal{L}\{u\} \cdot \mathcal{L}\{h\}

$$
```matlab
% 例：系统输入输出问题
% 已知：冲激响应 h(t) = (1/T)·e^(-t/T)·U(t)
%      输入信号 u(t) = e^(-t)·U(t)
% 求：系统输出 y(t) = u(t) * h(t)

clear all
syms T t tao s

% 定义输入和冲激响应
ut = exp(-t);                 % u(t) = e^(-t)
ht = exp(-t/T)/T;             % h(t) = (1/T)·e^(-t/T)

% 方法1：直接积分（时域卷积）
uh_tao = subs(ut, t, tao)*subs(ht, t, t - tao);  % 被积函数
yt1 = int(uh_tao, tao, 0, t)
yt1 = -(exp(-t) - exp(-t/T))/(T - 1)
yt1 = (exp(-t/T) - exp(-t))/(T - 1)  % 化简形式

% 方法2：使用拉普拉斯变换（频域相乘）
Ut = laplace(ut, t, s);
Ht = laplace(ht, t, s);
Yt = Ut * Ht;                 % 频域相乘

yt2 = ilaplace(Yt, s, t)
yt2 = exp(-t/T)/(T - 1) - exp(-t)/(T - 1)

% 化简并与方法1比较
yt22 = collect(yt2, (T - 1))
yt22 = (exp(-t/T) - exp(-t))/(T - 1)  % 完全相同！

% 绘制输出曲线
syms T_val
T_val = 2;                    % 取T=2

yt_plot = subs(yt1, T, T_val);
fplot(yt_plot, [0, 5])
grid on
xlabel('t')
ylabel('y(t)')
title('系统输出响应 y(t) = (e^{-t/T} - e^{-t})/(T-1)')
```


## 十二、Z变换[^1]

### 基本定义

**单边Z变换定义**（n从0开始）：
$$

F(z) = \sum_{n=0}^{\infty} f(n)z^{-n}

$$
**Z逆变换定义**（围线积分法）：
$$

f(n) = \frac{1}{2\pi j}\oint_{\Gamma} F(z)z^{n-1}dz

$$
**应用**：Z变换描述离散序列的复频率信息，可将差分方程转化为代数方程

### Z变换的计算[^1]

```matlab
clear
syms n z

% 例1：常数序列的Z变换
gn = 6*(1 - (1/2)^n)           % g(n) = 6[1 - (1/2)^n]
gn = 6 - 6*(1/2)^n

G = ztrans(gn, n, z);
pretty(G)
%  6 z       6 z
%  -----  -  -----
%  z - 1      z - 1/2

% 例2：正弦序列的Z变换
syms n w T z
fwn = sin(w*n*T);             % f(n) = sin(ω·T·n)
FW = ztrans(fwn, n, z);
pretty(FW)
%        z sin(T w)
%  ---------------------
%   2
%  z  - 2 cos(T w) z + 1

% 计算逆变换验证
inv_FW = iztrans(FW, z, n)
inv_FW = sin(T*n*w)           % 完美恢复！

% 例3：冲激序列的Z变换
% kroneckerDelta(n) 当n=0时为1，其余为0
delta = kroneckerDelta(n);
KD = ztrans(delta, n, z)
KD = 1                         % 冲激的Z变换为1

% 逆变换验证
inv_KD = iztrans(KD)
inv_KD = kroneckerDelta(n, 0)  % 恢复冲激

% 例4：时移性质
% 当序列仅在n=k处有值时
syms k positive
syms f(n)
assumeAlso(k, 'integer')       % k为正整数

fd = f(n)*kroneckerDelta(n - k);  % 仅f_d(k) = f(k)，其余为0
FD = ztrans(fd, n, z)
FD = f(k)/z^k                  % 时移性质：Z[f(n-k)] = z^(-k)F(z)

% 逆变换
inv_FD = iztrans(FD, z, n)
inv_FD = f(k)*kroneckerDelta(k - n, 0)

% 例5：指数序列的Z变换
syms a z n
GZ = exp(-a/z);               % G(z) = e^(-a/z)
gn = iztrans(GZ, z, n)
gn = (-a)^n/factorial(n)       % 结果为 (-a)^n/n!
```


### Z变换的常用性质[^1]

```matlab

% 1. 常数序列
ztrans(1, n, z)
ans = z/(z - 1)

% 2. 单位阶跃

ans = z/(z - 1)

% 3. 幂次序列

% 4. 指数序列

% 返回复杂的有理函数

% 5. 利用于线性系统

```


## 十三、三大变换的总结与比较[^1]

| 变换类型 | 定义域 | 频域变量 | 核函数 | 应用领域 |
| :-- | :-- | :-- | :-- | :-- |
| **傅里叶变换** | 连续时间 | ω（圆频率） | $e^{-j\omega t}$ | 连续信号分析、频谱分析 |
| **拉普拉斯变换** | 非负时间 | s（复频率） | $e^{-st}$ | 控制系统、稳定性分析 |
| **Z变换** | 离散时间 | z（复数） | $z^{-n}$ | 数字滤波、差分方程求解 |

### 三大变换的性质对应[^1]

```matlab
% 所有三大变换都满足类似的性质：

% 1. 线性性质：L[a·f(t) + b·g(t)] = a·L[f(t)] + b·L[g(t)]
% 2. 时域（或n域）导数/差分性质
% 3. 卷积定理：时域卷积 = 频域乘积
% 4. 时移（或n移）性质
% 5. 频移性质

% 例：MATLAB验证卷积定理
syms t s
f = exp(-t);
g = sin(t);

% 时域卷积
time_conv = int(subs(f, t, 't1')*subs(g, t, t - 't1'), 't1', [0, t]);

% 频域相乘
F = laplace(f, t, s);
G = laplace(g, t, s);
freq_mult = F*G;

% 逆变换频域乘积
result = ilaplace(freq_mult, s, t);

% 验证是否相等
err = simplify(time_conv - result);
% err应该等于0（误差来自化简顺序）
```

```matlab
disp('频域表达式：');
disp(F_w);

% 步骤3：化简
F_w_smp = simplify(F_w);
disp('化简后：');
disp(F_w_smp);                 % 2*A*sin(tau*w/2)/w

% 步骤4：逆变换验证
f_t_rec = ifourier(F_w_smp, w, t);
f_t_rec = simplify(f_t_rec);

% 步骤5：绘制时域和频域
% （代入具体参数）
A_val = 1; tau_val = 1;
F_w_plot = subs(F_w_smp, {A, tau}, {A_val, tau_val});

figure
subplot(2,1,1)
fplot(subs(f_t, {A, tau}, {A_val, tau_val}), [-2, 2])
title('时域信号')

subplot(2,1,2)
fplot(F_w_plot, [-20, 20])
title('频域信号（幅度谱）')
```

### 考前必背要点

**1. simplify选项**：

- `'Steps', n`：增加化简步数（常考！）
- `'IgnoreAnalyticConstraints', true`：激进化简

**2. subs用法**：

- 单变量：`subs(f, x, value)`
- 多变量：`subs(f, {x, y}, {val1, val2})`⭐⭐⭐
- 向量替换：`subs(f, x, [1, 2, 3])`生成向量

**3. 三大变换**：

| 变换 | 正变换 | 逆变换 | 应用领域 |
| :-- | :-- | :-- | :-- |
| Fourier | `fourier(f, t, w)` | `ifourier(F, w, t)` | 信号频谱 |
| Laplace | `laplace(f, t, s)` | `ilaplace(F, s, t)` | 控制系统 |
| Z变换 | `ztrans(f, n, z)` | `iztrans(F, z, n)` | 数字滤波 |
