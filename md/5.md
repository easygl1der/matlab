### funm：通用矩阵函数[^1]

```matlab
% funm(A, Hfun) = U·diag(f(λ₁), ..., f(λₙ))·U^(-1)
% Hfun为函数句柄

A = [1, 0; 0, 2];

% ❌ 错误写法：funm内部会传递额外参数给函数句柄
H_wrong = @(x) sin(x);        % 只接受1个参数
% F = funm(A, H_wrong)        % 报错：输入参数太多

% ✅ 正确写法1：使用内置函数名
F1 = funm(A, @sin)            % 直接使用@sin
F1 =
    0.8415         0
         0    0.9093

% ✅ 正确写法2：定义能处理多参数的函数句柄
H_correct = @(x, varargin) sin(x);  % 接受可变参数
F2 = funm(A, H_correct)       % 计算sin(A)
F2 =
    0.8415         0
         0    0.9093

% ✅ 正确写法3：使用function文件
% 创建文件 my_sin.m:
% function y = my_sin(x, varargin)
%     y = sin(x);
% end
% F3 = funm(A, @my_sin)

% 🔑 关键理解：
% funm内部调用时会传递额外参数：feval(fun, diag(T), 0, varargin{:})
% 所以函数句柄必须能接受这些额外参数，即使不使用它们

% 验证结果：对角矩阵的矩阵函数
% sin([1, 0; 0, 2]) = [sin(1), 0; 0, sin(2)]
%                   = [0.8415, 0; 0, 0.9093]
```

### 🎯 关键理解要点

**宽度的含义**：
- `%5d` → 总共占用5个字符位置
- 如果数字不够5位，左边补空格
- 如果数字超过5位，按实际长度显示

**浮点数格式的含义**：
- `%f` → 默认6位小数
- `%.2f` → 强制2位小数（四舍五入）
- `%8.2f` → 总宽度8个字符，其中小数部分2位

**记忆方法**：
- 宽度 = "这个数字要占多少个字符位置"
- 小数位 = "小数点后显示几位数字"

### 🔑 重要技巧和注意事项

1. **宽度控制**：`%5d` 表示总宽度为5，不足时用空格填充
2. **精度控制**：`%.3f` 表示小数点后3位
3. **组合控制**：`%8.2f` 表示总宽度8，小数点后2位
4. **左对齐**：`%-8.2f` 负号表示左对齐
5. **前导零**：`%05d` 用0填充而不是空格
6. **转义字符**：`\\`, `''`, `%%` 用于输出特殊字符本身

### 例：搜索函数的最小值[^1]

```matlab
% 例6.1-1：求 y = e^(-0.1x)·sin²(x) - 0.5·sin(x)·(x+0.1) 在 [-50,5] 的最小值

function [xmin, fmin, n] = exm060101(fx, a, b, Nt)
% 参数：
%   fx：函数句柄
%   a, b：区间端点
%   Nt：极小值重复次数阈值
% 返回：
%   xmin：最小值点
%   fmin：最小值
%   n：最终子区间分割数

[~, f0] = fminbnd(fx, a, b);  % 初始极小值
n = 1;
jj = 1;

while 1
    n = 2*n;                      % 子区间数加倍
    d = (b-a)/n;
    x = a:d:b;
    
    ii = 0;
    xc = zeros(1, n);
    fc = xc;
    
    for k = 1:n
        [w, f, eflag] = fminbnd(fx, x(k), x(k+1));
        if eflag > 0
            ii = ii + 1;
            xc(ii) = w;
            fc(ii) = f;
        end
    end
    
    xc = xc(1:ii);
    fc = fc(1:ii);
    [fmin, kk] = min(fc);
    xmin = xc(kk);
    
    % 停止条件判断
    if abs(f0 - fmin) < 1e-6
        jj = jj + 1;
        if jj > Nt
            break
        end
    elseif f0 - fmin > 1e-6
        f0 = fmin;
        jj = 1;
    end
end
end

% 调用示例
clear
fx = @(x)(sin(x)^2*exp(-0.1*x) - 0.5*sin(x)*(x+0.1));
a = -50; b = 5;
[xmin, fmin, n] = exm060101(fx, a, b, 3);

fprintf('在x = %6.5f处，函数到达最小值 %6.5f\n', xmin, fmin)
fprintf('最终子区间分割数为 %d\n', n)

% 输出：
% 在x = -19.60721处，函数到达最小值 -3.34765
% 最终子区间分割数为 128
```


## 七、模拟退火算法[^1]

### 算法原理

**模拟退火**是一种随机优化算法，通过模拟金属退火过程来求解全局最优值。主要思路：

1. 从初始点出发，进行随机扰动
2. 若新点函数值更小，则接受；否则以一定概率接受
3. 随着"温度"降低，接受差的解的概率逐渐降低
4. 最终收敛到最优解（或局部最优解）
```matlab
% 例：使用模拟退火求解前面的最小值问题
clear, close all

f = @(x)(sin(x)^2*exp(-0.1*x) - 0.5*sin(x)*(x+0.1));

% 初始化温度
ini_temperature = 300;
end_temperature = 1e-6;
temperature = ini_temperature;

% 初始搜索位置
x = rand(1)*55 - 50;         % 在[-50, 5]内随机

iter = 0;
energy = [];
xi = [];

% 主循环
while temperature > end_temperature
    energy_old = f(x);
    
    % 生成扰动（[-10, 10]范围，根据温度调整）
    delta = (rand(1)*20 - 10) * min(1, temperature/1e-4);
    
    % 确保扰动在定义域内
    if (x + delta < 5) && (x + delta > -50)
        temp = x + delta;
    else
        temp = x - delta;
    end
    
    energy_new = f(temp);
    delta_e = energy_new - energy_old;
    
    % Metropolis准则：接受更好的解，或以一定概率接受差的解
    if (delta_e < 0) || (exp(-delta_e/temperature) > rand())
        x = temp;
    end
    
    % 记录迭代过程
    iter = iter + 1;
```

## 十九、广度优先搜索（BFS）-跳马问题[^1]

### 跳马最少步数[^1]

```matlab
% 例6.1-4：象棋棋盘跳马最少步数问题
% 棋盘：10行9列
% 求每个位置的最少步数

function A = knight_steps()
% 返回10×9矩阵，每个位置存储从(1,2)出发的最少步数

rows = 10;
cols = 9;
A = -ones(rows, cols);      % -1表示未访问

% 马的8种可能移动
moves = [2,1; 2,-1; -2,1; -2,-1; 1,2; 1,-2; -1,2; -1,-2];

% 从(1,2)开始，步数为0
start_row = 1;
start_col = 2;
A(start_row, start_col) = 0;

% 使用队列实现BFS
queue = [start_row, start_col];  % 队列：存储[行, 列]
head = 1;
tail = 1;

while head <= tail
    % 取队列头部元素
    r = queue(head, 1);
    c = queue(head, 2);
    current_steps = A(r, c);
    head = head + 1;
    
    % 尝试8种移动
    for i = 1:8
        new_r = r + moves(i, 1);
        new_c = c + moves(i, 2);
        
        % 检查是否在棋盘内且未访问
        if (new_r >= 1) && (new_r <= rows) && ...
           (new_c >= 1) && (new_c <= cols) && ...
           (A(new_r, new_c) == -1)
            
            A(new_r, new_c) = current_steps + 1;
            
            % 加入队列
            tail = tail + 1;
            queue(tail, :) = [new_r, new_c];
        end
    end
end
end

% 调用并显示结果
A = knight_steps();
disp(A)
```

#### 考题1：Hilbert矩阵生成（⭐⭐⭐）

**题目**：生成n阶Hilbert矩阵 $H_{ij} = \frac{1}{i+j-1}$，对比循环和向量化方法的性能

**标准答案**：

```matlab
n = 500;

% 方法1：双重循环（低效）
tic
H1 = zeros(n, n);               % ⭐预分配
for i = 1:n
    for j = 1:n
        H1(i,j) = 1/(i+j-1);
    end
end
t1 = toc;

% 方法2：向量化（高效）⭐⭐⭐
tic
i = repmat((1:n)', 1, n);       % 列索引矩阵
j = repmat(1:n, n, 1);          % 行索引矩阵
H2 = 1./(i + j - 1);
t2 = toc;

% 验证与报告
fprintf('误差：%.2e\n', max(abs(H1(:)-H2(:))));
fprintf('方法1：%.4f秒\n', t1);
fprintf('方法2：%.4f秒\n', t2);
fprintf('加速比：%.0f倍\n', t1/t2);

% 评分标准：
% - 预分配空间：2分
% - 向量化实现：3分
% - 性能对比：2分
% - 结果验证：1分
```

#### 考题2：BFS跳马问题（⭐⭐ 算法题）

**关键代码框架**：

```matlab
function steps = knight_bfs(start_row, start_col)
    % 10×9棋盘，马的BFS最短路径
    
    % 初始化
    board = -ones(10, 9);           % -1表示未访问
    board(start_row, start_col) = 0; % 起点步数=0
    
    % 马的8种移动方式
    moves = [2,1; 2,-1; -2,1; -2,-1; 
             1,2; 1,-2; -1,2; -1,-2];
    
    % 队列（BFS核心）
    queue = [start_row, start_col];
    head = 1;
    
    while head <= size(queue, 1)
        r = queue(head, 1);
        c = queue(head, 2);
        current_step = board(r, c);
        head = head + 1;
        
        % 尝试8个方向
        for i = 1:8
            nr = r + moves(i, 1);
            nc = c + moves(i, 2);
            
            % 检查：在棋盘内且未访问
            if nr>=1 && nr<=10 && nc>=1 && nc<=9 && board(nr,nc)==-1
                board(nr, nc) = current_step + 1;
                queue = [queue; nr, nc];  % 入队
            end
        end
    end
    
    steps = board;
end

% 🔑 算法要点：
% - 队列实现BFS（先进先出）
% - 用-1标记未访问
% - 8个方向遍历
% - 边界检查很重要
```
