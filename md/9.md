## 一、M脚本与M函数基础[^1]

### 1.1 两种文件类型的区别[^1]

| 特性 | M脚本 | M函数 |
| :-- | :-- | :-- |
| 结构 | 仅有代码行 | `function [output]=fname(input)...end` |
| 参数 | 无参数列表 | 有参数和返回值 |
| 返回值 | 无 | 支持多个返回值 |
| 作用域 | 全局变量 | 局部变量作用域 |
| 调用方式 | 脚本名 | 函数名(参数) |
| 文件结尾 | 可省略end | 只有一个函数时可省略end |

### 1.2 M函数的标准格式[^1]

```matlab
function [output1, output2, ...] = fname(input1, input2, ...)
% H1行：一行简明描述（help时首先显示）
% 详细说明：
%   input1  说明
%   input2  说明
%   output1 说明
% 用法：
%   fname(...)
%   result = fname(...)
%   [out1,out2] = fname(...)
% Author: your_name, date

% 函数体
% ...

% 返回值通过给 output1, output2 赋值实现
% 如果只有一个函数，end 可省略
end
```

**可观察范例**：`edit ode45` 查看 MATLAB 内置函数的写法。[^1]

## 二、多输入/输出灵活函数设计[^1]

### 2.1 关键指令：nargin 与 nargout[^1]

- **nargin**：检查实际输入参数个数
- **nargout**：检查实际要求的输出参数个数

用于实现**参数个数可变**的函数。[^1]

### 2.2 例6.2-1：多边形绘制函数[^1]

```matlab
function [S, L] = exm060201(N, R, str)
% exm060201.m  正多边形的面积和周长
% 功能：根据边数、半径、颜色绘制正多边形
% 
% 输入：
%   N      边数（可省）
%   R      外接圆半径（可省）
%   str    颜色和线型（可省）
%
% 输出：
%   S      面积
%   L      周长
%
% 用法：
%   exm060201              % 默认：100条边，半径1，蓝色
%   exm060201(N)           % 指定边数
%   exm060201(N, R)        % 指定边数和半径
%   exm060201(N, R, str)   % 完整指定
%   S = exm060201(...)     % 仅返回面积
%   [S, L] = exm060201(...) % 返回面积和周长

% 根据输入参数个数设定默认值
switch nargin
    case 0
        N = 100; R = 1; str = '-b';
    case 1
        R = 1; str = '-b';
    case 2
        str = '-b';
    case 3
        % 无操作，三个参数均已提供
    otherwise
        error('输入参数过多');
end

% 生成多边形坐标
t = 0:2*pi/N:2*pi;
x = R * sin(t);
y = R * cos(t);

% 根据输出参数个数决定动作
if nargout == 0
    % 无输出：仅绘制空心多边形
    plot(x, y, str);
elseif nargout > 2
    % 输出过多：报错
    error('输出参数过多');
else
    % 有输出：计算面积和周长，绘制填色多边形
    S = N * R * R * sin(2*pi/N) / 2;      % 面积
    L = 2 * N * R * sin(pi/N);            % 周长
    fill(x, y, str);
end

% 绘图美化
axis image
set(gcf, 'Color', 'white')
box on
shg

% 调用示例
[S, L] = exm060201(6, 2, '-g')
% S = 10.3923, L = 12.0000（绘制绿色实心六边形）
```


## 三、主函数与子函数[^1]

### 3.1 文件结构[^1]

一个M文件可包含多个函数：

- **主函数**：第一个函数，文件名通常与函数名相同
- **子函数**：后续所有函数，**仅能被同文件内部调用**[^1]


### 3.2 作用域与调用方式[^1]

```
同文件内部：
  - 主函数可调用所有子函数
  - 脚本可调用所有函数
  - 子函数间可相互调用

外部访问：
  - 命令行可直接调用主函数
  - 无法直接调用子函数（需要通过主函数返回句柄）
```

**优点**：减少 M 文件数量，隐藏内部实现细节。[^1]

### 3.3 例6.3-1：主函数+子函数结构[^1]

```matlab
% 文件：exm060301.m
function Hr = exm060301(flag)
% exm060301.m  主函数与子函数演示
% 
% 输入：flag = 'line' 或 'circle'
% 输出：Hr = 子函数 cirline 的句柄

t = (0:50)/50 * 2*pi;           % 采样点
x = sin(t);
y = cos(t);

Hr = @cirline;                  % 返回子函数句柄
feval(Hr, flag, x, y, t);       % 调用子函数

% ========== 子函数定义 ==========
function cirline(wd, x, y, t)
% 子函数：绘制直线或圆形
% wd：'line' 或 'circle'
% x, y, t：坐标数据

switch wd
    case 'line'
        plot(t, x, 'b', t, y, 'r', 'LineWidth', 2)
    case 'circle'
        plot(x, y, '-g', 'LineWidth', 8)
        axis square off
    otherwise
        error('输入只能是 ''line'' 或 ''circle''')
end
shg

% 调用示例
HH = exm060301('circle')        % HH = @cirline
HH('line', x, y, t)            % 通过句柄间接调用子函数
```

**关键点**：子函数无法被外部直接调用，但可通过主函数返回的**函数句柄**在外部调用。[^1]

```matlab
% 查询句柄信息

% ans.function: 'magic'
```

### 4.2 句柄的调用方式[^1]

```matlab
% 方式1：直接调用
M1 = hm(4)                      % 等同于 magic(4)

% 方式2：feval间接调用
M2 = feval(hm, 4)               % 同样调用 magic(4)

% 方式3：作为参数传递给其他函数
```

## 五、一元函数极值问题[^1]

### 5.1 内置函数 fminbnd[^1]

```matlab
[x, fval, exitflag] = fminbnd(fun, x1, x2)
```

- `fun`：函数句柄或匿名函数
- `x1, x2`：搜索区间
- `x`：极小值点
- `fval`：极小值
- `exitflag > 0`：成功[^1]

**重点**：仅找到**局部最小值**，不一定是全局最小值。[^1]

### 5.2 例4.1-7：多极值问题[^1]

```matlab
% 求 y = e^(-0.1x)·sin²(x) - 0.5(x+0.1)sin(x) 在 [-50,5] 的最小值

x1 = -50; x2 = 5;
yx = @(x) sin(x).^2 .* exp(-0.1*x) - 0.5*sin(x).*(x+0.1);

% 粗探：快速扫描找局部极值
[xc0, fc0, exitflag] = fminbnd(yx, x1, x2)
% xc0 = -8.4867, fc0 = -1.8621

% 精细搜索：在疑似极值区间再次细搜
ezplot(yx, [-50, 5])            % 绘制函数图像
xx = [-23, -20, -18];           % 三个搜索区间端点
fc = fc0; xc = xc0;

for k = 1:2
    [xw, fw] = fminbnd(yx, xx(k), xx(k+1));
    if fw < fc
        xc = xw; fc = fw;       % 更新最小值
    end
end

fprintf('最小值 %.5f 发生在 x=%.5f\n', fc, xc)
% 最小值 -3.34765 发生在 x=-19.60721
```


## 六、单峰函数的黄金分割法[^1]

### 6.1 算法原理[^1]

对于单峰函数，在 $[a, b]$ 中选择两个试探点 $\xi_1, \xi_2$：

$$
\xi_1 = 0.618a + 0.382b, \quad \xi_2 = 0.382a + 0.618b
$$

比较 $f(\xi_1)$ 和 $f(\xi_2)$，**舍去较大值对应一侧**，不断缩小区间。

**时间复杂度**：$O(\ln\frac{b-a}{\epsilon})$

**优势**：处理不可导函数，无需计算导数。[^1]

### 6.2 代码实现[^1]

```matlab
% 求 y = e^(-0.1x)·sin²(x) - 0.5(x+0.1)sin(x) 在 [-20,-18] 的最小值

x1 = -20; x2 = -18;
yx = @(x) sin(x).^2 .* exp(-0.1*x) - 0.5*sin(x).*(x+0.1);

minf = min(yx(x1), yx(x2));     % 初始最小值

iter = 0;
while true
    iter = iter + 1;
    
    % 黄金分割点
    alpha1 = x1 * 0.618 + x2 * 0.382;
    alpha2 = x1 * 0.382 + x2 * 0.618;
    
    % 比较函数值，舍去较大值一侧
    if yx(alpha1) > yx(alpha2)
        x1 = alpha1;            % 排除左侧
        if yx(alpha2) < minf
            minf = yx(alpha2);
        end
    else
        x2 = alpha2;            % 排除右侧
        if yx(alpha1) < minf
            minf = yx(alpha1);
        end
    end
    
    % 收敛条件：区间足够小
    if alpha2 - alpha1 < 1e-8
        break;
    end
end

fprintf('最小值：%.5f，迭代次数：%d\n', minf, iter)
% 最小值：-3.34765，迭代次数：38
```


## 七、一元牛顿迭代法[^1]

### 7.1 求根问题[^1]

对方程 $f(x)=0$ 的求解，牛顿迭代公式：

$$
x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}
$$

### 7.2 求极值问题[^1]

对极小值问题，由于 $\arg\min f(x) \Leftrightarrow f'(x) = 0$，迭代公式改为：

$$
x_{n+1} = x_n - \frac{f'(x_n)}{f''(x_n)}
$$

```matlab
% 例：求 y = e^(-0.1x)·sin²(x) - 0.5(x+0.1)sin(x) 在 [-20,-18] 的最小值

syms x
yx = @(x) sin(x).^2 .* exp(-0.1*x) - 0.5*sin(x).*(x+0.1);
yxp = diff(yx, x);              % 一阶导数
yxp2 = diff(yxp, x);            % 二阶导数

x_old = -19;                    % 初值
iter = 0;

while true
    iter = iter + 1;
    % 牛顿迭代：x_new = x_old - f'(x) / f''(x)
    x_new = x_old - double(subs(yxp, x, x_old) / subs(yxp2, x, x_old));
    
    if abs(x_new - x_old) < 1e-8
        break;
    end
    x_old = x_new;
end

fprintf('极值点：x = %.5f，迭代次数：%d\n', x_new, iter)
% 极值点：x = -19.60721，迭代次数：5
minf = yx(x_new)
% minf = -3.3476
```

**优势**：二阶收敛，收敛很快（5次迭代 vs 黄金分割法38次）。[^1]

```matlab

```

- `x0`：初值（标量）
- `x`：零点
- `fval`：函数值（应接近0）[^1]

## 九、多元函数最优化[^1]

### 9.1 无约束优化：fminsearch[^1]

```matlab
[x, fval, exitflag, output] = fminsearch(fun, x0)
```

- `fun`：多元函数句柄，输入/输出为向量
- `x0`：初值向量
- `x`：极小值点向量
- `output.iterations`：迭代次数[^1]

```matlab
% 例：求 f(x,y) = 100(y-x²)² + (1-x)² 的极小值

ff = @(x) 100*(x(2) - x(1)^2)^2 + (1 - x(1))^2;

x0 = [-5, -2; -5, -2; 2, 2; 5, 5]';  % 4种初值
[sx, sfval, sexit, soutput] = fminsearch(ff, x0)
% 输出所有局部极值，第一个最接近全局最小值(1, 1)
```


### 9.2 更优函数：fminunc（推荐）[^1]

```matlab
options = optimoptions(@fminunc, 'OptimalityTolerance', 1e-8);
[x, fval] = fminunc(ff, x0, options)
```

- 基于**梯度信息**，收敛更快
- 精度更高
- 不易陷入局部极值（在梯度条件好的情况下）[^1]

```matlab
clear, clc
ff = @(x) 100*(x(2) - x(1)^2)^2 + (1 - x(1))^2;
x0 = [5, 5];

options = optimoptions(@fminunc, 'OptimalityTolerance', 1e-8);
[x, fval] = fminunc(ff, x0, options)
% x = [1.0000, 1.0000]
% fval = 2.0666e-11
```


## 十、多元函数梯度下降法[^1]

### 10.1 最速下降（梯度下降）法[^1]

对于凸函数，沿负梯度方向迭代：

$$
x^{(k+1)} = x^{(k)} + \lambda_k p^{(k)}, \quad p^{(k)} = -\nabla f(x^{(k)})
$$

其中 $\lambda_k$ 为**最优步长**（需求解一元优化）。[^1]

```matlab
% 例：求 f(x,y) = 100(y-x²)² + (1-x)² 的极小值

ff = @(x, y) 100*(y - x^2)^2 + (1 - x)^2;
dff = @(x, y) [2*x - 400*x*(-x^2 + y) - 2; -200*x^2 + 200*y];

x0 = -5; y0 = -2;
x_old = x0; y_old = y0;
iter = 0;

while true
    iter = iter + 1;
    
    % 计算梯度
    Grad = dff(x_old, y_old);
    
    % 在该方向上求最优步长（一元优化）
    lsf = @(lambda) ff(x_old - lambda*Grad(1), y_old - lambda*Grad(2));
    [lambda, ~] = fminbnd(lsf, 0, 10);
    
    % 梯度下降迭代
    x_new = x_old - lambda * Grad(1);
    y_new = y_old - lambda * Grad(2);
    
    if abs(x_new - x_old) < 1e-8 && abs(y_new - y_old) < 1e-8
        break;
    end
    x_old = x_new; y_old = y_new;
end

fprintf('极值点：(%.5f, %.5f)，迭代次数：%d\n', x_new, y_new, iter)
% 从 x0=-5, y0=-5：iter = 1641, x_new = 1.0000, y_new = 1.0000
```

**特点**：**收敛缓慢**（锯齿形），但对初值不敏感。[^1]

## 十一、多元函数牛顿法[^1]

### 11.1 Hesse矩阵与迭代公式[^1]

对多元函数 $f(x,y)$，Hesse矩阵为：

$$
\nabla^2 f = \begin{pmatrix} \frac{\partial^2 f}{\partial x^2} & \frac{\partial^2 f}{\partial x \partial y} \\ \frac{\partial^2 f}{\partial x \partial y} & \frac{\partial^2 f}{\partial y^2} \end{pmatrix}
$$

牛顿迭代公式：

$$
x^{(k+1)} = x^{(k)} - [\nabla^2 f(x^{(k)})]^{-1} \nabla f(x^{(k)})
$$

### 11.2 实现与对比[^1]

```matlab
% 相同问题，用牛顿法求解

ff = @(x, y) 100*(y - x^2)^2 + (1 - x)^2;
% 提前计算并编码 Hessian^{-1} · Gradient
hidff = @(x, y) [...
    (x - 1)/(200*x^2 - 200*y + 1), ...
    -(200*x^4 - 400*x^2*y - x^2 + 2*x + 200*y^2 - y)/(200*x^2 - 200*y + 1)
];

x0 = -5; y0 = -5;
x_old = x0; y_old = y0;
iter = 0;

while true
    iter = iter + 1;
    p = hidff(x_old, y_old);
    x_new = x_old - p(1);
    y_new = y_old - p(2);
    
    if abs(x_new - x_old) < 1e-8 && abs(y_new - y_old) < 1e-8
        break;
    end
    x_old = x_new; y_old = y_new;
end

fprintf('极值点：(%.5f, %.5f)，迭代次数：%d\n', x_new, y_new, iter)
% iter = 6（仅6次！vs 梯度下降法的1641次）
```

**对比表**：


| 方法 | 梯度下降 | 牛顿法 |
| :-- | :-- | :-- |
| 初值 (5,5) | 1115次 | 6次 |
| 初值 (-5,-5) | 1641次 | 6次 |
| 收敛速度 | 线性 | 二阶 |
| 计算复杂度 | 低 | 中等 |

牛顿法虽然计算 Hesse 矩阵昂贵，但迭代次数少得多。[^1]

## 十二、程序设计建议[^1]

### 推荐做法（课后反馈）[^1]

为黄金分割法、一元牛顿法、多元最速下降法、多元牛顿法各设计一个 M 函数，接口统一：

```matlab
function [xopt, fopt, iter] = goldensection(fun, a, b, tol)
% 黄金分割法
function [xopt, fopt, iter] = newton1d(fun, dfun, d2fun, x0, tol)

function [xopt, fopt, iter] = gradient_descent(fun, grad, x0, tol)

function [xopt, fopt, iter] = newton_multi(fun, grad, hess, x0, tol)
% 多元牛顿法
```

这样可方便复用于后续项目。[^1]

```matlab
% 正多边形面积和周长计算
% 
% 输入参数（可变）：
%   N     - 边数（默认100）
%   R     - 半径（默认1）

% 输出参数：
%   S - 面积
%   L - 周长

    % 处理可变输入参数⭐⭐⭐

        case 0
            N = 100; R = 1; color = 'b';
        case 1
            R = 1; color = 'b';
        case 2
            color = 'b';
        case 3
            % 所有参数都提供了
        otherwise
            error('参数过多');
    end
    
    % 计算
    S = N * R^2 * sin(2*pi/N) / 2;
    L = 2 * N * R * sin(pi/N);
    
    % 处理可变输出参数⭐⭐⭐
    if nargout == 0
        % 无输出：仅绘图
        theta = linspace(0, 2*pi, N+1);

        y = R*sin(theta);
        plot(x, y, color, 'LineWidth', 2);
        axis equal, grid on
    elseif nargout > 2
        error('输出参数过多');
    end
    % 有输出时自动返回S和L
end
```

#### 模板3：一元优化三种方法（⭐⭐⭐）

```matlab
% 问题：求f(x)在[a,b]的最小值

f = @(x) x.^2 - 4*x + 3;        % f(x) = x² - 4x + 3
a = 0; b = 5;

% 方法1：fminbnd（内置，最简单）⭐⭐⭐
[x1, fval1] = fminbnd(f, a, b);
fprintf('fminbnd：x=%.4f, f(x)=%.4f\n', x1, fval1);
% 输出：x=2.0000, f(x)=-1.0000

% 方法2：黄金分割法（算法题）
tol = 1e-6;
while b - a > tol
    alpha1 = a*0.618 + b*0.382;
    alpha2 = a*0.382 + b*0.618;
    
    if f(alpha1) > f(alpha2)
        a = alpha1;             % 舍弃左侧
    else
        b = alpha2;             % 舍弃右侧
    end
end
x2 = (a + b) / 2;
fprintf('黄金分割：x=%.4f, f(x)=%.4f\n', x2, f(x2));

% 方法3：牛顿法（需要导数）⭐⭐
syms x_sym
f_sym = x_sym^2 - 4*x_sym + 3;
fp = diff(f_sym);               % f'(x) = 2x - 4
fpp = diff(fp);                 % f''(x) = 2

x3 = 0;                         % 初值
for iter = 1:10
    x_new = x3 - double(subs(fp, x_sym, x3) / subs(fpp, x_sym, x3));
    if abs(x_new - x3) < 1e-8
        break;
    end
    x3 = x_new;
end
fprintf('牛顿法：x=%.4f, f(x)=%.4f, 迭代%d次\n', x3, f(x3), iter);

% 🔑 对比：
% - fminbnd：最方便，适合实际应用
% - 黄金分割：不需导数，适合不可导函数
% - 牛顿法：收敛最快（二阶），但需导数
```


#### 模板4：多元优化（⭐⭐⭐）

```matlab
% 问题：求f(x,y)的最小值（Rosenbrock函数）

% 定义函数（输入是向量）
f = @(X) 100*(X(2) - X(1)^2)^2 + (1 - X(1))^2;

% 方法1：fminsearch（无需梯度）
x0 = [0, 0];                    % 初值
[xmin1, fmin1] = fminsearch(f, x0);
fprintf('fminsearch：(%.4f, %.4f), f=%.2e\n', xmin1(1), xmin1(2), fmin1);

% 方法2：fminunc（基于梯度，更快）⭐⭐⭐
options = optimoptions(@fminunc, 'Display', 'off');
[xmin2, fmin2] = fminunc(f, x0, options);
fprintf('fminunc：(%.4f, %.4f), f=%.2e\n', xmin2(1), xmin2(2), fmin2);

% 理论最小值：(1, 1)，f(1,1) = 0
fprintf('理论值：(1.0000, 1.0000), f=0\n');

% 🔑 考试要点：
% - 函数输入是向量X，通过X(1), X(2)访问
% - 初值选择影响结果（局部极值）
% - fminunc比fminsearch更快更准
% - 多试几个初值找全局最小
```
