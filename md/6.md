## 一、数值导数[^1]

### 1.1 符号求导与雅可比矩阵[^1]

对矩阵表达式的**偏导数**，相当于对矩阵的每一个元素表达式分别求偏导。使用 `jacobian(f,v)` 可求得函数向量 $f$ 对自变量向量 $v$ 的**雅可比矩阵**，结果为符号表达式。[^1]

### 1.2 向前差分：diff 函数[^1]

`diff(X)` 计算向量或矩阵的向前差分：

$$
\text{dx} = X(2:n) - X(1:n-1)
$$

对矩阵：按行求差，后一行减前一行。结果长度比原矩阵少 1 行。[^1]

```matlab
X = [1, 3, 6, 10];
dx = diff(X)                    % dx = [2, 3, 4]

A = [1, 2, 3; 4, 5, 6; 7, 8, 9];
dA = diff(A)                    % 后一行减前一行
% dA = [3, 3, 3; 3, 3, 3]
```


### 1.3 中心差分：gradient 函数[^1]

`FX = gradient(F)` 实现**中心差分**运算：

- 内部点：$F_i^{(1)} = \frac{F_{i+1} - F_{i-1}}{2}$
- 端点：$F_1^{(1)} = F_2 - F_1$，$F_n^{(1)} = F_n - F_{n-1}$
- **长度与 F 相同**，精度高于向前差分[^1]

二维梯度 `[FX, FY] = gradient(F)`：

- `FX`：横向梯度（每行右减左）
- `FY`：纵向梯度（每列下减上）[^1]

```matlab
F = [1, 3, 6, 10];
FX = gradient(F)
% FX = [2, 2.5, 3.5, 4]（内部用中心差，端点用向前差）
```


### 1.4 例：sin 函数导数的近似[^1]

```matlab
% 例4.1-2、4.1-3：求 x(t) = sin(t) 在 [0,2π] 的近似导函数

d = pi/100;                     % 采样步长
t = 0:d:2*pi;
x = sin(t);

% ❌ 错误：增量太小会出现"除小毛刺"
dt = 5*eps;
x_eps = sin(t + dt);
dxdt_eps = (x_eps - x) / dt;   % 数值不稳定

% ✓ 合适：增量与采样步长同量级
x_d = sin(t + d);
dxdt_d = (x_d - x) / d;        % 合理的增量

% ✓ 使用diff（向前差分）
dxdt_diff = diff(x) / d;       % 长度少1

% ✓✓ 使用gradient（中心差分，推荐）
dxdt_grad = gradient(x) / d;   % 同长，精度更好

% 绘图对比
subplot(1,2,1)
plot(t, x, 'b', 'LineWidth', 2)
hold on
plot(t, dxdt_grad, 'm', 'LineWidth', 8)           % 品红色中心差分
plot(t(1:end-1), dxdt_diff, '.k', 'MarkerSize', 8) % 黑点向前差分
legend('x(t)', 'dxdt_{grad}', 'dxdt_{diff}', 'Location', 'North')
hold off
```

**关键经验**：

- 采样增量太小（$<\sim\text{eps}$）会导致"除小毛刺"，数值误差
- 最佳增量与采样步长同量级
- `gradient`（中心差分）精度优于 `diff`（向前差分）[^1]



## 三、数值积分[^1]

### 3.1 黎曼和与矩形公式[^1]

基于黎曼和定义：

$$
\int_a^b f(x)dx \approx \sum_{i=1}^n f(\xi_i) \Delta x_i
$$

其中 $\xi_i$ 可取左端点、右端点或中点。左端点精度最差。[^1]

### 3.2 梯形公式（Trapezoidal Rule）[^1]

对每个子区间 $[x_i, x_{i+1}]$，用梯形逼近：

$$
\int_{x_i}^{x_{i+1}} f(x)dx \approx \frac{f(x_i) + f(x_{i+1})}{2} \cdot (x_{i+1} - x_i)
$$

#### 手动实现：

```matlab
clear, d = 1e-5;               % 步长
x = 0:d:1;
fx = 1./(1 + x.^2);

% 计算每个梯形的底边高度平均值
fm = (fx(1:end-1) + fx(2:end)) / 2;

f_int = sum(fm) * d;           % 梯形求和
pi_app = f_int * 4
% pi_app = 3.1416, 误差 1.6669e-11

% 更细的步长
d = 1e-8;
x = 0:d:1;
fx = 1./(1 + x.^2);
fm = (fx(1:end-1) + fx(2:end)) / 2;
f_int = sum(fm) * d;
pi_app = f_int * 4
% pi_app ≈ 3.141592653589910（前12位小数可信）
```


### 3.3 trapz 函数详解[^1]

`trapz` 函数使用梯形公式计算数值积分，有三种主要语法：

#### **语法1：`Q = trapz(Y)` - 单位间距积分**

```matlab
% 假设采样点间距为1，只提供函数值
Y = [1, 4, 6, 8, 5];          % 函数值向量

Q = trapz(Y)                  % 默认间距dx=1
% 等价于：Q = 1 * trapz([0,1,2,3,4], Y)

% 🔑 关键理解：
% - 自动假设 X = [0, 1, 2, 3, 4]（单位间距）
% - 计算梯形面积：(1+4)/2*1 + (4+6)/2*1 + (6+8)/2*1 + (8+5)/2*1
% - 结果：Q = 2.5 + 5 + 7 + 6.5 = 21

% 实际应用：当你只有等间距的函数值时
t = 0:0.1:1;                  % 等间距时间点
v = sin(2*pi*t);              % 速度数据
distance = 0.1 * trapz(v);    % 总距离 = 步长 × trapz(速度)
```

#### **语法2：`Q = trapz(X, Y)` - 指定采样点积分**

```matlab
% 提供采样点X和对应的函数值Y
X = [0, 0.5, 1.2, 2.0, 3.1]; % 不等间距采样点
Y = [1, 4, 6, 8, 5];          % 对应的函数值

Q = trapz(X, Y)               % 使用实际的X坐标

% 🔑 关键理解：
% - 使用实际的X间距：[0.5, 0.7, 0.8, 1.1]
% - 梯形面积：(1+4)/2*0.5 + (4+6)/2*0.7 + (6+8)/2*0.8 + (8+5)/2*1.1
% - 结果：Q = 1.25 + 3.5 + 5.6 + 7.15 = 17.5

% 实际应用：实验数据通常不等间距
time = [0, 0.1, 0.3, 0.8, 1.5];     % 不规则时间点
temperature = [20, 25, 30, 28, 22];  % 温度测量值
heat_integral = trapz(time, temperature);  % 热量积分
```

#### **语法3：`Q = trapz(___, dim)` - 指定维度积分**

```matlab
% 对矩阵的指定维度进行积分
Y = [1, 2, 3;                 % 3×3矩阵
     4, 5, 6;
     7, 8, 9];

% 沿第1维（列方向）积分
Q1 = trapz(Y, 1)              % 结果：[12, 15, 18]（1×3向量）
% 每列分别积分：第1列=(1+4)/2 + (4+7)/2 = 2.5+5.5 = 8（实际是12）

% 沿第2维（行方向）积分  
Q2 = trapz(Y, 2)              % 结果：[4; 10; 16]（3×1向量）
% 每行分别积分：第1行=(1+2)/2 + (2+3)/2 = 1.5+2.5 = 4

% 🔑 关键理解：
% - dim=1：对每列进行积分（沿行方向求和）
% - dim=2：对每行进行积分（沿列方向求和）

% 实际应用：多通道数据处理
time = 0:0.1:2;                      % 时间向量
signals = [sin(time); cos(time); sin(2*time)]';  % 3通道信号（21×3矩阵）

% 计算每个通道的总能量
energy_per_channel = 0.1 * trapz(signals.^2, 1);  % 1×3向量

% 计算每个时刻的总功率
power_per_time = trapz(signals.^2, 2);             % 21×1向量
```

#### **完整对比示例**

```matlab
% 准备测试数据
x = [0, 1, 3, 6, 10];         % 不等间距
y = [2, 8, 18, 32, 50];       % 对应函数值 y=2x²

fprintf('=== trapz函数三种用法对比 ===\n');

% 方法1：只提供Y（假设单位间距）
Q1 = trapz(y);
fprintf('trapz(y) = %.2f\n', Q1);
fprintf('（假设x=[0,1,2,3,4]，单位间距）\n\n');

% 方法2：提供X和Y（实际间距）
Q2 = trapz(x, y);
fprintf('trapz(x,y) = %.2f\n', Q2);
fprintf('（使用实际x坐标）\n\n');

% 方法3：矩阵维度积分
Y_matrix = [y; 2*y; 3*y];     % 3×5矩阵
Q3_dim1 = trapz(Y_matrix, 1); % 沿第1维
Q3_dim2 = trapz(Y_matrix, 2); % 沿第2维
fprintf('矩阵积分（dim=1）：[%.1f, %.1f, %.1f, %.1f, %.1f]\n', Q3_dim1);
fprintf('矩阵积分（dim=2）：[%.1f; %.1f; %.1f]\n', Q3_dim2);

% 🎯 选择建议：
% - 等间距数据 → trapz(Y)
% - 不等间距数据 → trapz(X, Y)  
% - 多维数据 → trapz(..., dim)
```

#### **与理论积分对比**

```matlab
% 验证精度：计算∫₀¹⁰ 2x² dx = [2x³/3]₀¹⁰ = 2000/3 ≈ 666.67

% 粗糙采样
x_coarse = [0, 5, 10];

% 精细采样

% 理论值

fprintf('粗糙采样：%.2f，误差：%.2f%%\n', Q_coarse, abs(Q_coarse-theoretical)/theoretical*100);
fprintf('精细采样：%.2f，误差：%.2f%%\n', Q_fine, abs(Q_fine-theoretical)/theoretical*100);
```

### 3.4 辛普森公式（Simpson's Rule）[^1]

对每个子区间，用二次多项式逼近：

$$
\int_{x_i}^{x_{i+1}} f(x)dx \approx \frac{f(x_i) + 4f(x_i+h/2) + f(x_{i+1})}{6} \cdot h
$$

其中 $h = x_{i+1} - x_i$。

```matlab
clear, d = 1e-5;               % 全步长
x = 0:d:1;
xh = x + d/2;                  % 半步长采样点

fx = 1./(1 + x.^2);
fxh = 1./(1 + xh.^2);

% 代入辛普森公式
fm = (fx(1:end-1) + 4*fxh(1:end-1) + fx(2:end)) / 6;

f_int = sum(fm) * d;
pi_app = f_int * 4
% pi_app = 3.1416，误差 1.33239e-15（前14位可信！）

format long, pi_app, format
% pi_app = 3.141592653589794
```

**精度对比**（同步长 $d=10^{-5}$）：

- **梯形公式**：误差 $\sim 10^{-11}$，前10位可信
- **辛普森公式**：误差 $\sim 10^{-15}$，前14位可信
- **辛普森精度远优于梯形**，相同步长可减少100倍以上误差[^1]


### 3.5 蒙特卡洛法（Monte Carlo Integration）[^1]

#### 基本思想：

$$
\int_a^b f(x)dx \approx \frac{\sum_{j=1}^K f(x_j)}{K} \cdot (b-a)
$$

其中 $x_j \in [a,b]$ 为**随机采样点**，$K$ 通常不少于 $10^4$。[^1]

#### 一维实现：

```matlab
clear, Nsp = 1e5;              % 采样点数
x = rand(1, Nsp);              % [0,1]上均匀分布

fx = 1./(1 + x.^2);
f_int = sum(fx) / Nsp;         % 求平均
pi_app = f_int * 4
% pi_app ≈ 3.1421，误差 ≈4.8739e-04

% 每次运行结果不同（随机采样）
```


#### 二维蒙特卡洛：估计 $\pi$[^1]

在 $[0,1] \times [0,1]$ 正方形内随机投 $N$ 个点，统计落在圆 $(x-0.5)^2 + (y-0.5)^2 \leq 0.25$ 内的点数 $N_a$，则

$$
\pi \approx \frac{4N_a}{N}
$$

**特点**：

- 优点：处理离散数据和高维积分简单
- 缺点：误差大（$\sim O(1/\sqrt{K})$），不适合被积函数局部绝对值大的情况
- 适用场景：已有未知位置的随机采样数据[^1]


## 四、匿名函数[^1]

适用于简单的"一句话函数"，无需创建 m 文件：

$$
\text{F} = @(\text{参数列表}) \text{函数主体}
$$

```matlab
% 单参数
F = @(x) sin(x) .* x;
F(3)                           % 直接调用

% 多参数
G = @(x, y) x.^2 + y.^2;
G(3, 4)                        % 返回25

% 向量化（必须用点运算）
H = @(x) x.^2;
H([1, 2, 3])                   % 返回[1, 4, 9]

% 矩阵值函数
M = @(x) x.^[-0.2; 5];         % 返回2×1列向量
M(0.5)

% 作为参数传递
feval(F, 5)                    % 间接调用
```


## 五、MATLAB 内置数值积分函数 `integral`[^1]

### 5.1 基本用法[^1]

```matlab
% Q = integral(fun, xmin, xmax)
% 默认绝对误差上限：1e-10，相对误差上限：1e-6
% 使用自适应辛普森法、自适应罗巴托法等

Q = integral(@(x) sin(x).^2, 0, pi)

% 指定精度
Q = integral(@(x) 1./(1+x.^2), 0, 1, 'AbsTol', 1e-12)
```


### 5.2 阵列值函数[^1]

当被积函数返回向量时，需要 `'ArrayValued', true`：

```matlab
% 错误：不指定阵列值选项
G = @(x) x.^[-0.2; 5];        % 返回2×1列向量
% Q = integral(G, 0, 1)       % ❌ 报错或错误结果

% 正确：使用阵列值选项
Q = integral(G, 0, 1, 'ArrayValued', true)
% Q = [积分1; 积分2]
```


### 5.3 路径积分（复变函数）[^1]

```matlab
% 沿指定路径积分
F = @(z) (2*z - 1) / (z*(z - 1));
Path = [2+1i, -1+1i, -1-1i, 2-1i, 2+1i];
sf = integral(F, 2+1i, 2+1i, 'Waypoints', Path)
```


### 5.4 广义积分[^1]

```matlab

% 瑕积分（无需特殊处理）

```


## 六、曲线夹面积问题（例4.1-5）[^1]

求 $g_1(x) = \frac{1}{x^{0.2}}$ 与 $g_2(x) = x^5$ 在 $x \in [0,1]$ 夹面积：

```matlab
format long

G1 = @(x) x.^-0.2;
Q1 = integral(G1, 0, 1)
% Q1 = 1.250000027856048

G2 = @(x) x.^5;
Q2 = integral(G2, 0, 1)
% Q2 = 0.166666666666667

S12 = Q1 - Q2
% S12 = 1.083333361189381

% 符号计算验证
syms t
Gsym = vpa(int(t.^[-0.2; 5], 0, 1));
Ssym = Gsym(1) - Gsym(2)
% Ssym = 1.0833333333333333333333333333333（31位精度）

% 阵列方法一行代码
G = @(x) x.^[-0.2; 5];
Q = integral(G, 0, 1, 'ArrayValued', true)
S = [1, -1] * Q                 % 矩阵运算

% 🤔 为什么两种方法结果不同？精度分析：

% 方法1：分别计算（较低精度）
% Q1 = 1.250000027856048  ← 数值积分误差
% Q2 = 0.166666666666667  ← 数值积分误差  
% S12 = 1.083333361189381 ← 两个误差累积

% 方法2：阵列值计算（较高精度）
% Q = integral(G, 0, 1, 'ArrayValued', true)
% 内部使用相同的采样点同时计算两个积分，减少了误差累积

% 符号计算（理论精确值）
% Ssym = 1.0833333333333333333333333333333（31位精度）

% 🔑 关键差异：
% 1. 分别计算：两次独立的数值积分，误差会累积
% 2. 阵列计算：同一组采样点计算，误差更一致，相减时部分抵消
% 3. 符号计算：解析求解，理论上无数值误差

% 详细误差分析：
format long
fprintf('分别计算误差：%.15e\n', abs(S12 - 13/12));
fprintf('阵列计算误差：%.15e\n', abs(S - 13/12));
fprintf('理论值：%.15f\n', 13/12);

% 🎯 实际应用建议：
% - 需要高精度：使用符号计算 vpa(int(...))
% - 多个相关积分：使用 'ArrayValued', true
% - 单个积分：直接使用 integral
```


## 七、二重积分的复合中矩体公式[^1]

矩形区域 $D=[a,b] \times [c,d]$ 上，分成 $m \times n$ 个小正方形，每个取中点：

$$
\iint_D f(x,y) \, dxdy \approx \sum_{i=1}^m \sum_{j=1}^n f(\xi_i, \eta_j) \cdot S_{\square}
$$

```matlab
% 例：计算 ∬_D 1/√((1+x²+y²)³) dxdy，D:[0,1]×[0,1]

f = @(x, y) 1./sqrt((1 + x.^2 + y.^2).^3);

d = 1/2000;                    % 步长
x = d/2 : d : 1 - d/2;         % 中矩体采样点
y = x;

% 关键：x为列向量，y为行向量，使得 f(x',y) 形成2000×2000矩阵
intmr = sum(sum(f(x', y))) * d * d

% 结果：intmr = 0.5236
% 理论值：π/6 ≈ 0.52359877559829887
% 误差：abs(pi/6 - intmr) ≈ 8.0188e-09
```

**关键细节**：

- `x'` 变成 2000×1 列向量
- `y` 保持 1×2000 行向量
- MATLAB 自动广播扩展为 2000×2000 矩阵
- `sum(sum(...))` 两层求和得标量[^1]

### 核心代码模板

#### 模板1：数值导数对比（⭐⭐⭐ 必考）

```matlab
% 任务：计算sin(x)的导数，对比三种方法

% 准备数据
d = pi/100;                     % 采样步长
x = 0:d:2*pi;
f = sin(x);

% 方法1：diff（向前差分）
df1 = diff(f) / d;              % 长度 = length(f)-1
x1 = x(1:end-1);                % 对应的x坐标

% 方法2：gradient（中心差分）⭐⭐⭐ 推荐
df2 = gradient(f) / d;          % 长度 = length(f)
x2 = x;                         % x坐标不变

% 方法3：手动增量法
dx = d;
f_plus = sin(x + dx);
df3 = (f_plus - f) / dx;        % 长度 = length(f)
x3 = x;

% 理论导数
df_theory = cos(x);

% 误差分析
err1 = norm(df1 - df_theory(1:end-1));
err2 = norm(df2 - df_theory);
err3 = norm(df3 - df_theory);

fprintf('diff误差：%.2e\n', err1);
fprintf('gradient误差：%.2e\n', err2);
fprintf('手动增量误差：%.2e\n', err3);

% 🔑 考试要点：
% - gradient精度最高（中心差分）
% - diff长度会减少1
% - 增量太小（<eps）会导致数值不稳定
% - 增量应与步长同量级
```


#### 模板2：数值积分方法对比（⭐⭐⭐）

```matlab
% 任务：计算∫₀¹ 1/(1+x²) dx = π/4

% 方法1：梯形公式（trapz）
d = 1e-5;
x = 0:d:1;
y = 1./(1 + x.^2);
Q1 = trapz(x, y);               % 或 d*trapz(y)
err1 = abs(Q1 - pi/4);

% 方法2：辛普森公式（手动，高精度）⭐⭐
x = 0:d:1;
xh = x + d/2;                   % 半步点
y = 1./(1 + x.^2);
yh = 1./(1 + xh.^2);
ym = (y(1:end-1) + 4*yh(1:end-1) + y(2:end)) / 6;
Q2 = sum(ym) * d;
err2 = abs(Q2 - pi/4);

% 方法3：integral函数（最方便）⭐⭐⭐
Q3 = integral(@(x) 1./(1+x.^2), 0, 1);
err3 = abs(Q3 - pi/4);

% 方法4：蒙特卡洛（低精度）
N = 1e6;                        % 采样点数
x_rand = rand(1, N);
y_rand = 1./(1 + x_rand.^2);
Q4 = mean(y_rand) * 1;          % 区间长度×平均值
err4 = abs(Q4 - pi/4);

% 对比
fprintf('方法          步长/点数    误差\n');
fprintf('梯形          %.0e     %.2e\n', d, err1);
fprintf('辛普森        %.0e     %.2e\n', d, err2);
fprintf('integral      自适应      %.2e\n', err3);
fprintf('蒙特卡洛      %.0e点    %.2e\n', N, err4);

% 🔑 结论：
% - 辛普森精度最高（同步长下）
% - integral最方便（自适应步长）
% - 蒙特卡洛适合离散随机数据
```


#### 模板3：匿名函数应用（⭐⭐⭐）

```matlab
% 基本语法：F = @(参数) 表达式

% 单参数
f1 = @(x) sin(x).*x;
```

### 考前必记公式

**1. 数值导数**：

- 向前差分：$f'(x_i) \approx \frac{f(x_{i+1}) - f(x_i)}{h}$
- 中心差分：$f'(x_i) \approx \frac{f(x_{i+1}) - f(x_{i-1})}{2h}$ ⭐精度更高

**2. 数值积分**：

- 梯形公式：$\int_a^b f(x)dx \approx \frac{h}{2}[f(x_0) + 2f(x_1) + \cdots + 2f(x_{n-1}) + f(x_n)]$
- 辛普森公式：$\int_a^b f(x)dx \approx \frac{h}{6}[f(x_i) + 4f(x_{i+1/2}) + f(x_{i+1})]$
