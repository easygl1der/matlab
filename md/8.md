### 1.2 例：符号矩阵的基本操作[^1]

```matlab

% 其中 D = sqrt((A11-A22)^2 + 4*A12*A21)
```

```matlab
% 方法1：矩阵除法（推荐，速度快）
A = sym([1, 1/2, 1/2, -1;

eq4 = -8*d - n + p + q - 1;

```


### 2.2 无穷多解的情况[^1]

当方程组欠定（3个方程，4个变量）时：

```matlab
syms d n p q
eq1 = d + n/2 + p/2 - q;
eq2 = n + d + q - p - 10;
eq3 = q + d - n/4 - p;

S = solve(eq1, eq2, eq3, d, n, p, q);
disp([S.d, S.n, S.p, S.q])
% 新版本提供一个特解：[-2, 8, -4, 0]
% 旧版本可能提供含有参数的通解形式
```

**关键**：在无穷多解情况下，新版 MATLAB 只给出特解，完整通解需手动计算或使用 `null` 函数。[^1]

## 三、矩阵范数[^1]

### 3.1 常见范数的定义与计算[^1]

#### 2-范数（谱范数）[^1]

$$
\|A\|_2 = \max_{\|x\|_2=1} \|Ax\|_2 = \sqrt{\lambda_{\max}(A^T A)} = \sigma_{\max}(A)
$$

```matlab
norm(A)         % 计算2-范数，等同于norm(A,2)
```


#### Frobenius范数[^1]

$$
\|A\|_F = \sqrt{\sum_i (A^T A)_{ii}} = \sqrt{\text{trace}(A^T A)}
$$

常用于矩阵误差分析。

```matlab
norm(A, 'fro')  % Frobenius范数
```


#### 1-范数（列范数）[^1]

$$
\|A\|_1 = \max_{1 \leq j \leq n} \sum_{i=1}^m |a_{ij}|
$$

```matlab
norm(A, 1)      % 列范数：最大列绝对值之和
```


#### ∞-范数（行范数）[^1]

$$
\|A\|_\infty = \max_{1 \leq i \leq m} \sum_{j=1}^n |a_{ij}|
$$

```matlab
norm(A, inf)    % 行范数：最大行绝对值之和
```


#### 0-范数（非零元素个数）[^1]

$$
\|A\|_0 = \#\{(i,j): a_{ij} \neq 0\}
$$

```matlab
sum(sum(A ~= 0.0))  % 0-范数，不是真正的范数
```

衡量矩阵**稀疏性**的直接指标。[^1]

#### 核范数（奇异值之和）[^1]

$$
\|A\|_* = \sum_{i=1}^n \sigma_i(A)
$$

```matlab
[U, S, V] = svd(A);
norm(diag(S), 1)    % 核范数：奇异值的1-范数
```

用于**低秩逼近与低秩求解**。[^1]

## 四、低秩约束优化[^1]

### 4.1 三种低秩模型[^1]

**1. 强制降秩模型**（最直接）[^1]

$$
\min_{A^*} \|A^*-B\|_F \quad \text{s.t.} \quad \text{rank}(A^*) \leq r
$$

**2. 低秩惩罚项模型**（自动控制秩）[^1]

$$
\min_{A^*} \|A^*-B\|_F^2 + \lambda \cdot \text{rank}(A^*)
$$

**3. 松弛化低秩模型**（用核范数代替秩）[^1]

$$
\min_{A^*} \|A^*-B\|_F^2 + \lambda \cdot \|A^*\|_*
$$

**4. 半正定核范数模型**（当 $A^* \succeq 0$ 时）[^1]

$$
\min_{A^*} \|A^*-B\|_F^2 + \lambda \cdot \text{trace}(A^*)
$$

其中 $\lambda > 0$ 是调控参数。[^1]

## 五、矩阵的标量特征[^1]

### 5.1 秩、行列式、迹[^1]

```matlab
% 例：reshape生成测试矩阵
A = reshape(1:9, 3, 3)
% A = [1, 4, 7; 2, 5, 8; 3, 6, 9]

r = rank(A)                     % 秩为2（三行线性相关）
d3 = det(A)                     % det(A) = 0
d2 = det(A(1:2, 1:2))           % 子矩阵行列式 = -3
t = trace(A)                    % 迹 = 1+5+9 = 15

% 迹的性质：trace(AB) = trace(BA)
tAB = trace(A*B)
tBA = trace(B*A)                % tAB == tBA（即使AB ≠ BA）

% 行列式的性质
det(A*B) == det(A)*det(B)       % |AB| = |A|·|B|
det(B*A) == det(B)*det(A)       % |BA| = |B|·|A|，交换不变

% 特殊：非方阵相乘时行列式可能不同
det(C*D) % 结果为 d1
det(D*C) % 结果为 d2，可能 d1 ≠ d2
```

**关键性质**：

- $\text{trace}(AB) = \text{trace}(BA)$ 对同阶方阵或"内维"相等矩阵成立
- $\det(AB) = \det(BA) = \det(A) \cdot \det(B)$ 仅对同阶方阵成立[^1]


## 六、矩阵变换与空间[^1]

### 6.1 行阶梯型化简[^1]

```matlab
A = magic(4)
% A = [16, 2, 3, 13; 5, 11, 10, 8; 9, 7, 6, 12; 4, 14, 15, 1]

[R, ci] = rref(A)               % 化为行阶梯型
% R = [1, 0, 0, 1; 0, 1, 0, 3; 0, 0, 1, -3; 0, 0, 0, 0]
% ci = [1, 2, 3]   % 阶梯元素列指标，表示 rref(A) 中每个主元（阶梯元素）对应A的第几列

r_A = length(ci)                % 秩 = 3

% 利用行变换关系表示A的第4列
aa = A(:, 1:3) * R(1:3, 4)      % R(1:3,4) = [1; 3; -3]
% aa = A(:,1) + 3*A(:,2) - 3*A(:,3) = A(:,4)
```

**关键性质**：初等行变换不改变列向量的线性相关性，因此可用相同系数表示原矩阵的列。[^1]

### 6.2 零空间（核）与值空间（像）[^1]

```matlab
% 零空间：Ax = 0 的所有解构成的空间
X = null(A)                     % 返回零空间的标准正交基

% 验证：AX = 0（误差为数值舍入）
S = A * X
% S = [2.7e-14; ...; 6.2e-14]（接近零）

% 秩的另一定义：rank(A) + dim(null(A)) = n
r_A = length(ci);
l = size(X, 2);
n - l == r_A                    % 返回 1（成立）

% 值空间：A的列向量张成的空间
Y = orth(A)                     % 返回值空间的标准正交基
```

## 八、线性方程组求解[^1]

### 8.1 左除运算符 vs inv[^1]

```matlab
% 高条件数病态矩阵示例
rng default
A = gallery('randsvd', 300, 2e13, 2);  % cond(A) ≈ 2e13
x = ones(300, 1);
b = A * x;

% 方法1：inv（不推荐）
tic
xi = inv(A) * b;
ti = toc                        % ti ≈ 0.0273
eri = norm(x - xi)              % eri ≈ 0.0957
rei = norm(A*xi - b) / norm(b)  % rei ≈ 0.0055（相对误差大）

% 方法2：左除（推荐）
tic
xd = A \ b;
td = toc                        % td ≈ 0.0110（更快）
erd = norm(x - xd)              % erd ≈ 0.0290
red = norm(A*xd - b) / norm(b)  % red ≈ 8.37e-15（更精确！）
```

**结论**：对于病态矩阵，**A\b** 不仅更快，而且回代相对误差更小。永远优于 `inv(A)*b`。[^1]

### 8.2 欠定与超定问题[^1]

```matlab
A = reshape(1:12, 4, 3);        % 4×3矩阵（列满秩）
b = (13:16)';

ra = rank(A)                    % ra = 2
rab = rank([A, b])              % rab = 2
% rank(A) == rank([A,b]) < n，存在无穷多解

xs = A \ b;                     % 一个特解
xg = null(A);                   % 零空间基（齐次通解）

% 验证：任意特解 + 齐次特解 都满足方程
c = rand(1);
ba = A * (xs + c*xg);
ba == b                         % 成立
```


## 九、迭代法求解线性方程组[^1]

### 9.1 雅可比迭代法[^1]

分裂 $A = D - L - U$，令 $M=D, N=L+U$：

$$
x^{(k+1)} = Bx^{(k)} + f = D^{-1}(L+U)x^{(k)} + D^{-1}b
$$

```matlab
rng default
A = rand(400, 400) + 5e2*eye(400);  % 对角占优矩阵
x_true = rand(400, 1);
b = A * x_true;

D = diag(diag(A));              % 提取对角元
N = D - A;                      % N = L + U
B = D \ N;                      % B = D^{-1}·N
f = D \ b;

x_old = zeros(400, 1);
err = 1e12;
iter = 0;

while err > 1e-6
    iter = iter + 1;
    x_new = B * x_old + f;
    err = norm(x_new - x_old);
    x_old = x_new;
end

fprintf('迭代次数: %d\n', iter);       % iter = 19
fprintf('最终误差: %.4e\n', norm(x_new - x_true));
```

**收敛条件**：$\|B\| < 1$，需要矩阵对角占优。[^1]

### 9.2 高斯-塞德尔迭代法[^1]

分裂方式：$M = D-L, N = U$：

$$
x^{(k+1)} = (D-L)^{-1}Ux^{(k)} + (D-L)^{-1}b
$$

收敛要求比雅可比法更低（只需A正定）。实现类似，但循环中的 $x_i^{(k+1)}$ 立即参与计算。[^1]

## 十、基于变分法的梯度方法[^1]

### 10.1 最速梯度下降法（详细步骤）[^1]

对于对称正定矩阵 $A$，线性方程组 $Ax=b$ 可以转化为二次型最小化问题：

$$
\Phi(x) = \frac{1}{2} x^T A x - b^T x
$$

使用最速梯度下降法求解，具体步骤如下：

$$
-\nabla \Phi(x^{(k)}) = b - Ax^{(k)} = p^{(k)}
$$
$$
\lambda_k = \frac{\langle p^{(k)}, p^{(k)} \rangle}{\langle Ap^{(k)}, p^{(k)} \rangle}
$$
2. **循环迭代，直到收敛（即 $||x^{(k+1)} - x^{(k)}|| < 10^{-6}$）**：

    1. 令 $p^{(k)} = b - A x^{(k)}$，这是当前点的负梯度方向（也是下降最快的方向）。
    
    2. 计算最优步长 $\lambda_k$，公式如下：
       $$
       \lambda_k = \frac{(p^{(k)})^T p^{(k)}}{(A p^{(k)})^T p^{(k)}}
       $$
       这样选取可以使目标函数 $\Phi(x)$ 在该方向下降最快。
    
    3. 沿最速方向更新解：$x^{(k+1)} = x^{(k)} + \lambda_k p^{(k)}$
    
    4. 计算收敛标准 $err = ||x^{(k+1)} - x^{(k)}||$，更新 $x^{(k)} \gets x^{(k+1)}$。

    5. 迭代计数器 `iter = iter + 1`。

3. **输出最终结果**：输出迭代步数与与真解的误差。

下面为详细的 MATLAB 实现：

```matlab
% 步骤1：初始化
rng default
A0 = rand(300, 300);
A = A0 + A0' + 1e3 * eye(300);  % 构造对称正定矩阵
x_true = rand(300, 1);          % 随机生成真解
b = A * x_true;                 % 计算右端项
x_old = zeros(300, 1);          % 初始猜测 x^(0)
err = 1e12;                     % 初始误差很大，以保证能进入循环
iter = 0;                       % 迭代计数器

% 步骤2：逐步迭代
while err > 1e-6
    iter = iter + 1;
    
    % 2.1 计算当前负梯度方向
    p = b - A * x_old;
    
    % 2.2 计算最优步长
    lambda = (p' * p) / ((A * p)' * p);
    
    % 2.3 沿最速方向更新解
    x_new = x_old + lambda * p;
    
    % 2.4 计算收敛标准
    err = norm(x_new - x_old);
    
    % 2.5 更新
    x_old = x_new;
end

% 步骤3：输出最终结果
fprintf('迭代次数: %d\n', iter);                   % 例如 iter = 8
fprintf('最终误差: %.4e\n', norm(x_new - x_true)); % 查看收敛精度
```

**算法流程总结**：

1. 初始化 $x^{(0)}$；
2. $p^{(k)} = b - A x^{(k)}$；
3. 计算步长 $\lambda_k = \frac{(p^{(k)})^T p^{(k)}}{(A p^{(k)})^T p^{(k)}}$；
4. 更新 $x^{(k+1)} = x^{(k)} + \lambda_k p^{(k)}$；
5. 重复2-4, 直到收敛。

**特点**：精度比雅可比法高，但收敛缓慢（震荡）。[^1]

### 10.2 共轭梯度法（CG）[^1]

共轭梯度法选择方向 $p^{(k)}$ 使其为 A-共轭，收敛速度显著提升：

```matlab
rng default
A0 = rand(300, 300);
A = A0 + A0' + 1e3*eye(300);
x_true = rand(300, 1);
b = A * x_true;

x_old = zeros(300, 1);
r_old = b - A * x_old;          % 初始残差
p = r_old;                      % 初始方向
err = 1e12;
iter = 0;

while err > 1e-6
    iter = iter + 1;
    lambda = (r_old' * r_old) / ((A*p)' * p);
    x_new = x_old + lambda * p;
    err = norm(x_new - x_old);
    r_new = r_old - lambda * A * p;  % 残差更新
    eta = (r_new' * r_new) / (r_old' * r_old);  % 方向修正系数
    p = r_new + eta * p;        % 共轭方向更新
    x_old = x_new;
    r_old = r_new;
end

fprintf('迭代次数: %d\n', iter);       % iter = 5
fprintf('最终误差: %.4e\n', norm(x_new - x_true));
```

**优势**：对N维正定问题，最多N次迭代收敛（理论），远优于最速下降法。[^1]

### 10.3 使用内置函数 `pcg`[^1]

```matlab
% 直接使用预条件共轭梯度法
x_new = pcg(A, b);              % 默认误差容限 1e-6
% 输出：converged at iteration 4 with relative residual 3.3e-08

x_new2 = pcg(A, b, 1e-12);      % 更严格的容限
% 输出：converged at iteration 7 with relative residual 1.1e-14
```

**优点**：自动处理迭代终止条件，支持函数句柄（避免超大矩阵乘法）。[^1]

## 十二、本章核心要点[^1]

1. **范数类型**：2-范数（谱）、Frobenius、1-范数（列）、∞-范数（行）、0-范数（稀疏）、核范数（低秩）
2. **左除优于求逆**：对病态矩阵，`A\b` 回代误差更小
3. **迭代法的收敛条件**：需 $\|B\| < 1$ 或矩阵对角占优/正定
4. **共轭梯度法效率最高**：对对称正定问题，最多 n 步收敛
5. **零空间与秩**：$\text{rank}(A) + \dim(\text{null}(A)) = n$
6. **行阶梯型的作用**：揭示列向量的线性表示关系

| **迭代法** | ⭐⭐ | Jacobi、Gauss-Seidel | 算法实现 |

#### 模板2：线性方程组求解（⭐⭐⭐ 必考）

```matlab
% 问题：求解Ax = b

A = [1 2 3; 4 5 6; 7 8 10];     % 3×3满秩矩阵
b = [1; 2; 3];

% 方法1：左除（最推荐！）⭐⭐⭐
x1 = A \ b;

% 方法2：求逆矩阵（不推荐）
x2 = inv(A) * b;

% 方法3：使用rref（手动求解）
[R, pivots] = rref([A, b]);
x3 = R(:, end);

% 性能和精度对比
fprintf('左除解：'); disp(x1');
fprintf('求逆解：'); disp(x2');
fprintf('rref解：'); disp(x3');

% 回代误差（衡量精度）
residual1 = norm(A*x1 - b);
residual2 = norm(A*x2 - b);
fprintf('左除回代误差：%.2e\n', residual1);
fprintf('求逆回代误差：%.2e\n', residual2);

% 🔑 考试结论：
% - 永远用A\b，不用inv(A)*b
% - A\b更快（尤其大矩阵）
% - A\b更准（病态矩阵时显著）
% - 记住："左除求解线性方程"
```


#### 模板3：迭代法求解（⭐⭐ 算法题）

```matlab
% Jacobi迭代法：x^(k+1) = D^(-1)(L+U)x^(k) + D^(-1)b

% 构造对角占优矩阵
rng(0);
A = randn(100, 100) + 100*eye(100);  % 对角占优⭐
x_true = randn(100, 1);
b = A * x_true;

% 分解：A = D - L - U
D = diag(diag(A));              % 对角阵
L_plus_U = D - A;               % L+U

% 迭代矩阵
B = D \ L_plus_U;               % D^(-1)·(L+U)
f = D \ b;                      % D^(-1)·b

% Jacobi迭代
x = zeros(100, 1);              % 初值
max_iter = 1000;
tol = 1e-6;

for iter = 1:max_iter
    x_new = B*x + f;
    
    % 收敛判断
    if norm(x_new - x) < tol
        fprintf('迭代%d次后收敛\n', iter);
        break;
    end
    x = x_new;
end

% 验证
err = norm(x - x_true);
fprintf('与真实解误差：%.2e\n', err);

% 🔑 考试要点：
% - 对角占优才能保证收敛
% - 迭代矩阵B = D^(-1)(L+U)
% - 收敛条件：||B|| < 1
% - Gauss-Seidel比Jacobi快
```


#### 模板4：共轭梯度法（⭐⭐）

```matlab
% 共轭梯度法求解对称正定系统Ax = b

% 使用pcg函数（最简单）⭐⭐⭐
x = pcg(A, b, 1e-10, 100);      % 容差1e-10，最多100次

% 手动实现
x = zeros(n, 1);
r = b - A*x;                    % 初始残差
p = r;                          % 初始方向
tol = 1e-6;

for iter = 1:n
    Ap = A*p;
    alpha = (r'*r) / (p'*Ap);   % 步长
    x = x + alpha*p;            % 更新解
    
    r_new = r - alpha*Ap;       % 更新残差
    
    if norm(r_new) < tol
        fprintf('迭代%d次收敛\n', iter);
        break;
    end
    
    beta = (r_new'*r_new) / (r'*r);  % 方向修正
    p = r_new + beta*p;         % 更新方向
    r = r_new;
end

% 🔑 考试要点：
% - 仅对对称正定矩阵有效
% - 最多n步精确收敛（理论）
% - 实际3-10步即可达到高精度
% - pcg比手动实现更稳定
```


### 考前必背公式

**1. 矩阵范数定义**：

- $\|A\|_1 = \max_j \sum_i |a_{ij}|$ （列和最大）
- $\|A\|_\infty = \max_i \sum_j |a_{ij}|$ （行和最大）
- $\|A\|_2 = \sigma_{\max}(A)$ （最大奇异值）
- $\|A\|_F = \sqrt{\sum_{i,j} a_{ij}^2}$ （Frobenius范数）

**2. 迭代法收敛条件**：

- Jacobi/Gauss-Seidel：需对角占优或正定
- 共轭梯度：需对称正定
- 收敛条件：$\|B\| < 1$

**3. 秩的性质**：

- $\text{rank}(A) + \dim(\text{null}(A)) = n$
- $\text{rank}(AB) \leq \min(\text{rank}(A), \text{rank}(B))$
